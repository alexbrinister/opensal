Complex Split-to-Interleaved Conversion Functions

Complex split-to-interleaved conversion functions convert complex interleaved data to complex split data.

Table 5-5: Complex Split-to-Interleaved Conversion Functions

Function Description
cvcombdx Vector combine, double (create complex vector)
cvcombx Vector combine (create complex vector)
ztocdx Split to interleaved complex, double
ztocx Split to interleaved complex

Complex Vector Generate from Real and Imaginary Parts

cvcombx(), cvcombdx()

Syntax

void cvcombx(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *B, /* input vector */
  SAL_i32 j, /* address stride for B */
  SAL_cf32 *C, /* output vector */
  SAL_i32 K, /* address stride for C, K=2k */
  SAL_i32 N, /* complex element count */
  SAL_i32 eflag ); /* ESAL flag */
void cvcombdx(
  SAL_f64 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f64 *B, /* input vector */
  SAL_i32 j, /* address stride for B */
  SAL_cf64 *C, /* output vector */
  SAL_i32 K, /* address stride for C, K=2k */
  SAL_i32 N, /* double-complex element count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The cvcombx() and cvcombdx() functions create a complex vector C by combining real vectors A and B. These functions are the inverse of cvsplitx() and cvsplitdx().

Copy Split Complex Vector to Interleaved Complex Vector

ztocx(), ztocdx()

Syntax

extern void ztocx(
  SAL_zf32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_cf32 *C, /* output vector */
  SAL_i32 k, /* address stride for C, must be even */
  SAL_i32 N, /* complex element count */
  SAL_i32 eflag ); /* ESAL flag */
void ztocdx(
  SAL_zf64 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_cf64 *C, /* output vector */
  SAL_i32 k, /* address stride for C, must be even */
  SAL_i32 N, /* complex element count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The ztocx() and ztocdx() functions copy the contents of a split complex vector to an interleaved complex vector.
