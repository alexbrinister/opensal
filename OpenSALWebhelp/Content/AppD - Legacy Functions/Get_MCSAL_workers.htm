Get number of workers used by last MCSAL function

sal_get_n_workers_used() 

Syntax

SAL_i32 sal_get_n_workers_used(

SAL_handle h_sal,

SAL_i32 *p_n_workers_used

);

Description

The sal_get_n_workers_used() function gets the number of workers used by the last MCSAL function that was executed.

 
The sal_get_n_workers_used() function has been replaced by sal_get_n_threads_used().

If a data set is too small, and if using fewer workers will result in better performance, the MCSAL function might not use all of the workers available to it.

If your application defines the SAL_ENABLE_MULTICORE macro, the sal_get_n_workers_used() function returns the number of workers that were used by the last MCSAL function executed.

For example, on a dual-core processor, if the data is not large enough, MCSAL invokes the single-core version of the function on the manager thread. This function returns the SAL_SINGLE_USED value for the number of workers used.

Parameters

h_sal input

The SAL_handle to modify.

*p_n_workers_used output

The number of workers used by the last MCSAL function.

Return Codes

SAL_SUCCESS

Successful operation. If the operation is not successful, an error code is returned.
