Matrix Multiply

mmulx(), mmuldx()

Syntax

void mmulx(
  SAL_f32 2 *A, /* input matrix */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *B, /* input matrix */
  SAL_i32 j, /* address stride for B */
  SAL_f32 *C, /* output matrix */
  SAL_i32 k, /* address stride for C */
  SAL_i32 M, /* row count for A and C */
  SAL_i32 N, /* column count for B and C */
  SAL_i32 P, /* column count for A, row count for B */
  SAL_i32 eflag ); /* ESAL flag */
void mmuldx(
  SAL_f64 *A, /* input matrix */
  SAL_i32 i, /* address stride for A */
  SAL_f64 *B, /* input matrix */
  SAL_i32 j, /* address stride for B */
  SAL_f64 *C, /* output matrix */
  SAL_i32 k, /* address stride for C */
  SAL_i32 M, /* row count for A and C */
  SAL_i32 N, /* column count for B and C */
  SAL_i32 P, /* column count for A, row count for B */
  SAL_i32 eflag ); /* ESAL flag */

Description

The mmulx() and mmuldx() functions multiply M x P matrix A by P x N matrix B and store the results in M x N matrix C. These functions can be done out-of-place only, and provide better performance when multiplying two matrices using address strides of 1.
