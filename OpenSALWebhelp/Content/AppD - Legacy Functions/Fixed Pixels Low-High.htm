Fixed Pixels, Low-High Format

fixpixax()

Synopsis

void fixpixax(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_ui8 *C, /* 4-byte-aligned output vector */
  SAL_i32 N, /* real input count */
  SAL_i32 flag ); /* ESAL flag */

Description

The fixpixax() function converts two 32-bit elements at a time, packing pairs of 8-bit results into single 16-bit locations.

 
The fixpixax() function is not recommended for new application development. Use the newer vconvert_f32_u8x() function.
