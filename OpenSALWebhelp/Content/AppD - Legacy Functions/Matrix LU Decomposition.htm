Matrix LU Decomposition

matludx(), matluddx(), cmatludx(), cmatluddx(), zmatludx(), zmatluddx()

C = L • U

Description

The matludx(), cmatludx(), and zmatludx() functions perform an LU decomposition of a square matrix. The matluddx(), cmatluddx(), and zmatluddx() functions are the double-precision versions.

 
The matludx(), cmatludx(), and zmatludx() functions are not recommended for new application development. Use the mat_lud_decx(), cmat_lud_decx(), zmat_lud_decx() functions for matrix LU decomposition.

Output matrix C consists of two matrices:

  • L (occupies the lower left half of C)
  • U (occupies the upper right half of C)

The diagonal elements of the submatrix L are assumed to be 1 and not stored in C.

The output matrix overwrites the input matrix with a packed format. Vector D is a permutation vector created by the function.

Example: (for N = 4)Press, William H., Brian P. Flannery, Saul A. Teukolsky, William T. Vetterling, Numerical Recipes in C (New York: Cambridge University Press 1997), p. 43-48. ISBN 0-521-43108-5.1
