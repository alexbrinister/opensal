Vector Integer to Real Convert

vflt32x(), vfltu32x()

Cnk = float Ani        n = {0, N - 1}

Synopsis

void vflt32x(
  SAL_i32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 N, /* real output count */
  SAL_i32 flag ); /* ESAL flag */
void vfltu32x(
  SAL_ui32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 N, /* real output count */
  SAL_i32 flag ); /* ESAL flag */

Description

The vflt32x() and vfltu32x() functions convert 32-bit signed and unsigned, respectively, integers from vector A to real values. These functions can be done out-of-place only.

 
The vflt32x() and vfltu32x() functions are not recommended for new application development. Use the newer vconvert_s32_f32x() and vconvert_u32_f32x() functions for vector conversion.
