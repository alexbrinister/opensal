Vector Real to Nearest Integer or Rounded Short Convert

vnintx(), vnint2x()

Cnk = fix(Ani)          n = {0, N – 1}

Synopsis

void vnintx(
  SAL_f32 *A, /* pointer to input vector */
  SAL_i32 i, /* address stride for A */
  SAL_i32 *C, /* pointer to output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 N, /* real input count */
  SAL_i32 flag ); /* ESAL flag */
void vnint2x(
  SAL_f32 *A, /* pointer to input vector */
  SAL_i32 i, /* address stride for A */
  SAL_i16 *C, /* pointer to output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 N, /* real input count */
  SAL_i32 flag ); /* ESAL flag */

Description

The vnintx() function converts real input values ranging from –2,147,483,648 to 2,147,483,647, inclusive, from vector A to 32-bit signed integers, rounded toward the nearest integer.

The vnint2x() function converts real input values ranging from –32,768 to 32,767 inclusive from vector A to 16-bit signed integers, rounded toward the nearest integer.

If two integers are equally near, the integer with the greatest magnitude is output. Results for input values outside of this range are undefined.

 

The vnintx() and vnint2x() functions are not recommended for new application development. Use the newer vconvert_f32_s32x() and vconvert_f32_s16x() functions, respectively, with scale=NULL, bias=NULL, and rfflag=SAL_ROUND_NEAREST.
