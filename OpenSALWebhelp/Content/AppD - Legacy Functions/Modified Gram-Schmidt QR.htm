Description

The matmgs_srx(), cmatmgs_srx(), and zmatmgs_srx() functions solve triangular matrix systems of n equations in n unknowns Rx = b produced by the MGS QR decomposition functions. These functions accept an n x n input triangular matrix R and a right-hand side vector of known values B, and solve for the unknowns using back-substitution.

The matmgs_srhrx(), cmatmgs_srhrx(), and zmatmgs_srhrx() functions also solve triangular matrix systems of n equations in n unknowns RHRx = b produced by the MGS QR decomposition functions.

 

The matmgs_srx(), cmatmgs_srx(), zmatmgs_srx(), matmgs_srhrx(), cmatmgs_srhrx(), and zmatmgs_srhrx() functions are not recommended for new application development. Use the mat_mgs_solrx(), mat_mgs_solrhrx(), cmat_mgs_solrx(), cmat_mgs_solrhrx(), zmat_mgs_solrx(), and zmat_mgs_solrhrx() functions for matrix MGS QR solvers.

In addition, these functions accept an n x n input matrix R and a right-hand side vector of known values b, but solve for the unknowns by first solving RHy = b for y by forward substitution, and then Rx = y for x by back substitution. Both halves of the special packed R matrix are used in this case.

 
Matrix RH is the conjugate transpose of a complex matrix R, also known as the Hermitian conjugate.

All of these MGS QR solvers require the special R matrix format. The complete MGS QRD functions, matmgs_dqrx(), cmatmgs_dqrx(), and zmatmgs_dqrx(), produce the special packed R matrix format required by the solver functions. When using the matmgs_dkerx(), cmatmgs_dkerx(), and zmatmgs_dkerx() functions for parallel implementation of the OR decomposition, the MGS QR solvers require that you use one of the auxiliary R-transpose functions matmgs_trx(), cmatmgs_trx(), or zmatmgs_trx().
