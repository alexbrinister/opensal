Float Pixels, Low-High Format

fltpixax()

Synopsis

void fltpixax(
  SAL_ui8 *A, /* 4-byte-aligned input vector */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 N, /* real output count */
  SAL_i32 flag /* ESAL flag */
  );  

Description

The fltpixax() function converts a vector of unsigned 8-bit integers in low-high format into real values. This function can only be done out-of-place.

 
The fltpixax() function is not recommended for new application development. Use the modern vconvert_u8_f32x() function for vector conversion.
