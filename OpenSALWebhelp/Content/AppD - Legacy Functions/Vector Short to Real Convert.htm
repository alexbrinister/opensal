Vector Short to Real Convert

vfltx(), vfltux()

Synopsis

void vfltx(
  SAL_i16 *A, /* 4-byte aligned input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 n, /* real output count, must be even */
  SAL_i32 flag ); /* ESAL flag */
void vfltux(
  SAL_ui16 *A, /* 4-byte aligned input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 n, /* real output count, must be even */
  SAL_i32 flag ); /* ESAL flag */

Description

The vfltx() and vfltux() functions convert 16-bit signed and unsigned integers, respectively, from vector A to real values.

 
The vfltx() and vfltux() functions are not recommended for new application development. Use the newer vconvert_s16_f32x() and vconvert_u16_f32x() functions for vector conversion with unit strides.

These functions unpack and convert 16-bit input two at a time from 32-bit locations. They can be done out-of-place only.
