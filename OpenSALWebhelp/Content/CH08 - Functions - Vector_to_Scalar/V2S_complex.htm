Vector-to-Scalar Functions, Complex

Complex vector-to-scalar functions operate on complex data vectors.

Table 7-2: Complex Vector-to-Scalar Functions

Complex Interleaved Complex Interleaved Double Complex Split Complex Split Double Description
cdotprx cdotprdx zdotprx zdotprdx Vector dot product
cidotprx cidotprdx zidotprx zidotprdx Vector inner dot product
crdotprx crdotprdx zrdotprx zrdotprdx Vector dot product, double real

Vector Dot Product

cdotprx(), cdotprdx(), zdotprx(), zdotprdx()

Syntax

void cdotprx(
  SAL_cf32 *A, /* input vector */
  SAL_i32 I, /* address stride for A, I=2i */
  SAL_cf32 *B, /* input vector */
  SAL_i32 J, /* address stride for B, J=2j */
  SAL_cf32 *c, /* output scalar */
  SAL_i32 N, /* complex element count */
  SAL_i32 eflag ); /* ESAL flag */
void cdotprdx(
  SAL_cf64 *A, /* input vector */
  SAL_i32 I, /* address stride for A, I=2i */
  SAL_cf64 *B, /* input vector */
  SAL_i32 J, /* address stride for B, J=2j */
  SAL_cf64 *c, /* output scalar */
  SAL_i32 N, /* double-complex element count */
  SAL_i32 eflag ); /* ESAL flag */
void zdotprx(
  SAL_zf32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_zf32 *B, /* input vector */
  SAL_i32 j, /* address stride for B */
  SAL_zf32 *c, /* output scalar */
  SAL_i32 N, /* complex element count */
  SAL_i32 eflag ); /* ESAL flag */
void zdotprdx(
  SAL_zf64 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_zf64 *B, /* input vector */
  SAL_i32 j, /* address stride for B */
  SAL_zf64 *c, /* output scalar */
  SAL_i32 N, /* complex element count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The cdotprx(), cdotprdx(), zdotprx(), and zdotprdx() functions calculate the complex dot product of vectors A and B, and leave the sum in scalar c.
