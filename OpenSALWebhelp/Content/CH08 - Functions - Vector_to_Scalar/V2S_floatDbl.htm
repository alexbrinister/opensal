Vector-to-Scalar Functions, Float and Double Precision

Vector Dot Product

dotprx(), dotprdx()

Syntax

void dotprx(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *B, /* input vector */
  SAL_i32 j, /* address stride for B */
  SAL_f32 *c, /* output scalar */
  SAL_i32 N, /* real element count */
  SAL_i32 eflag ); /* ESAL flag */
void dotprdx(
  SAL_f64 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f64 *B, /* input vector */
  SAL_i32 j, /* address stride for B */
  SAL_f64 *c, /* output scalar */
  SAL_i32 N, /* double-precision element count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The dotprx() and dotprdx() functions compute the dot (or scalar) product of vectors A and B and store the sum in scalar c.

Vector Maximum Magnitude with Index

maxmgvix()  

Syntax

void maxmgvix(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *c, /* output scalar */
  SAL_i32 *d, /* output scalar */
  SAL_i32 N, /* real input count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The maxmgvix() function copies the element with the greatest magnitude from real vector A to real scalar c, and writes its zero-based index to integer scalar d. The index reflects address stride i. If vector A contains more than one instance of c, d contains the index of the first instance of c.

Find Zero Crossing

nzcrosx()

Syntax

void nzcrosx(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_i32 b, /* input scalar */
  SAL_i32 *c, /* output scalar, index */
  SAL_i32 *d, /* output scalar, zero crossings */
  SAL_i32 N, /* real input count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The nzcrosx() function scans vector A to locate transitions from positive to negative values and from negative to positive values. The scan terminates when the number of crossings that scalar b specifies is found.

The zero-based index of the specified crossing b is returned in scalar c. Index c reflects address stride i. If the zero crossing that b specifies is not found, zero is returned in c. The total number of zero crossings found is returned in d.
