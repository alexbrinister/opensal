Introduction to the Scientific Algorithm Library

The Scientific Algorithm Library (SAL) provides a set of 1-D (vector) and 2-D (matrix) mathematical functions.

SAL functions operate on integer, real, and complex (interleaved and split) data types.

SAL functions perform:

  • Matrix operations
  • Fast Fourier transforms (FFTs)
  • Data conversions
  • Signal processing and image analysis
  • Vector-to-vector and vector-to-scalar operations
  • Vector comparisons
