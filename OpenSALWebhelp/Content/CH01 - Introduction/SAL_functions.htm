SAL Library Functions

This document provides calling conventions and operational descriptions for the SAL functions. Each description includes the function's ANSI C prototype, usually annotated with comments.

In formulas and prototypes:

  • Uppercase letters represent vector and matrix parameters
  • Lowercase letters represent scalar parameters and flags

The SAL Header File

To use SAL functions, add the following line to your list of #include statements:

#include <sal.h>

If a function requires another header file, it is indicated in the function description.
