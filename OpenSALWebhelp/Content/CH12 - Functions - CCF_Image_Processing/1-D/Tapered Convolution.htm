Tapered Convolution or Correlation with Lag

tconvlx()

Syntax

void tconvlx(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *B, /* input filter coefficients, length of
B = q-p+1 */
  SAL_i32 j, /* address stride for B */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 m, /* real output count */
  SAL_i32 n, /* input element count for A */
  SAL_i32 p, /* input scalar, low summation limit,
p <= q */
  SAL_i32 q, /* input scalar, high summation limit */
  SAL_i32 eflag ); /* ESAL flag */

Description

The tconvlx() function performs:

  • Convolution if the signs of i and j differ

    - or - 

  • Correlation if the signs of i and j are the same

Zeros are substituted for inputs from vector A when the computed offset m + q is outside the range of vector A, specified by n. If address stride i is negative, A should point to the last vector element. If address stride j is negative, B should point to the last vector element. This function can be done out-of-place only.
