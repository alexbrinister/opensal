Difference Equation, 2 Poles, 2 Zeros

deq22x()

Syntax

void deq22x(
  SAL_f32 *A, /* input vector, length of A >= n + 2 */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *B, /* 5 inputs, filter coefficients */
  SAL_f32 *C, /* output vector, length of C >= n + 2 */
  SAL_i32 k, /* address stride for C */
  SAL_i32 N, /* real output count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The deq22x() function performs two-pole, two-zero recursive filtering on real input vector A. Because the computation is recursive, the first two elements in vector C must be initialized prior to calling the deq22x() function. The deq22x() function creates n new values for vector C beginning with its third element and requires N + 2 input values from vector A. This function can be done out-of-place only.
