Convolution Filter, Three Taps

fir3x()

 

Syntax

void fir3x(
  SAL_f32 *A, /* input vector */
  SAL_f32 *B, /* three filter coefficients */
  SAL_f32 *C, /* partial sums vector */
  SAL_f32 *D, /* output vector */
  int k, /* address stride for D */
  int N, /* number of output points */
  int f, /* partial sums flag: 0 (no sums) or 1 (add sums) */
  int eflag ); /* ESAL flag */

Description

The fir3x() function filters input vector a with a 3-element coefficients vector. If parameter f equals 1, the function uses input vector C as a preexisting partial-sums vector. Each partial sum in C is added to its corresponding output point in vector D. If parameter f equals zero, vector C is not used. Other values of f are undefined.

The fir3x() function partial-sum feature supports efficient implementations of short filters with various tap sizes using a combination of multiple calls to fir3x(), or to fir3x() and fir2x(). Output vectors with each call are fed into the next call as partial sum vectors. A 5-tap filter, for instance, can be implemented with a call to fir3x() followed by a call to fir2x(). For the second call, output vector D from the first call is used as the partial sum vector C. To produce results equivalent to a dedicated 5-tap filter function, the pointers to signal vector A and coefficients vector B are incremented by 3.
