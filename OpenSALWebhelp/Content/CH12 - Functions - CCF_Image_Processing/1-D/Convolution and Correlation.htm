Convolution and Correlation

convx(), cconvx(), zconvx()

Syntax

void convx(
  SAL_f32 *A, /* input vector, length of A >= N+p-1 */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *B, /* input filter */
  SAL_i32 j, /* address stride for B */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 N, /* real output count */
  SAL_i32 p, /* filter length (vector B) */
  SAL_i32 eflag ); /* ESAL flag */
void cconvx(
  SAL_cf32 *A, /* input vector */
  SAL_i32 I, /* address stride for A, I=2i */
  SAL_cf32 *B, /* input filter */
  SAL_i32 J, /* address stride for B, J=2j */
  SAL_cf32 *C, /* output vector */
  SAL_i32 K, /* address stride for C, K=2k */
  SAL_i32 N, /* real output count */
  SAL_i32 P, /* filter length (vector B); P <= 512 */
  SAL_i32 eflag ); /* ESAL flag */
void zconvx(
  SAL_zf32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_zf32 *B, /* input filter */
  SAL_i32 j, /* address stride for B */
  SAL_zf32 *C, /* output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 N, /* real output count */
  SAL_i32 P, /* filter length (vector B); P <= 512 */
  SAL_i32 eflag ); /* ESAL flag */

Description

The convx(), cconvx(), and zconvx() functions perform correlation or convolution on two vectors. If j is positive, they perform correlation. If j is negative, they perform convolution, and filter B must point to the last vector element.

These functions can run in-place, but C cannot be in-place with B.
