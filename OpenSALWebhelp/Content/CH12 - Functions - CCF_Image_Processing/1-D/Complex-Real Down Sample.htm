Complex/Real Down Sample with Anti-aliasing

desampx(), crdesampx(), zrdesampx()

Syntax

void desampx(
  SAL_f32 *A, /* Input vector */
  SAL_i32 f, /* Decimation factor */
  SAL_f32 *B, /* Filter coefficient vector */
  SAL_f32 *C, /* Output vector */
  SAL_i32 N, /* Length of output vector */
  SAL_i32 P, /* Coefficient count vector B */
  SAL_i32 eflag ); /* ESAL flag */
void crdesampx (
  SAL_cf32 *A, /* Input vector. */
  SAL_i32 f, /* Complex decimation factor */
  SAL_f32 *B, /* Filter coefficient vector, = 2*(decimation factor */
  SAL_cf32 *C, /* Output vector */
  SAL_i32 N, /* Length of output vector */
  SAL_i32 P /* Length of filter vector B */
  SAL_i32 eflag ); /* ESAL flag */
void zrdesampx(
  SAL_zf32 *A, /* Split complex input vector */
  SAL_i32 f, /* Decimation factor */
  SAL_f32 *B, /* Filter coefficient vector */
  SAL_zf32 *C, /* Output vector */
  SAL_i32 N, /* Length of output vector */
  SAL_i32 P, /* Length of filter vector B */
  SAL_i32 eflag ); /* ESAL flag */

Description

The desampx(), crdesampx(), and zrdesampx() functions perform finite impulse response (FIR) filtering with decimation of the output.

For interleaved complex vectors, the complex decimation factor is twice the decimation factor. For example, if input vector A (declared as SAL_cf32 *A) is undecimated, then f = 2. If A is decimated by a factor of two, then f = 4, and so on. This is analogous to interleaved complex vector strides, which are measured in terms of half elements.

The length of A must be at least (N-1) + D + P, where D is the decimation factor, (i.e., D = f for desampx and zrdesampx, and D = f/2 for crdesampx).

These functions can run in-place, but C cannot be in-place with B.
