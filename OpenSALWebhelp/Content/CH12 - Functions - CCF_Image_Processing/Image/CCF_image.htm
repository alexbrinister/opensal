Image Processing

Image processing functions perform a variety of specialized two-dimensional image processing and filtering operations, such as back-projection of a vector onto a 2-D image.

Table 11-4: 2-D Image Processing Functions, Float and Double

Float Double Description
bprojimgx bprojimgdx Back projection of single projection over image
bprojrowx bprojrowdx Image back projection over single row
imgfirx imgfirdx Image filter, m x n convolution (listed under Two-Dimensional Correlation, Convolution, and Filtering)
spinx spindx Image rotate, scale and translate with interpolation
warpx warpdx Image transformation with interpolation
