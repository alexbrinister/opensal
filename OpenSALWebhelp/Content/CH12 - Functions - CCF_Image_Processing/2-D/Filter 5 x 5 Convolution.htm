Filter, 5 x 5 Convolution

f5x5x(), f5x5dx()

Syntax

void f5x5x(
  SAL_f32 *A, /* input image */
  SAL_i32 M, /* row count for A and C */
  SAL_i32 N, /* column count for A and C */
  SAL_f32 *B, /* 5x5 input matrix of coefficients */
  SAL_f32 *C /* output image */
  SAL_i32 eflag ); /* ESAL flag */
void f5x5dx(
  SAL_f64 *A, /* input image */
  SAL_i32 M, /* row count for A and C */
  SAL_i32 N, /* column count for A and C */
  SAL_f64 *B, /* 5x5 input matrix of coefficients */
  SAL_f64 *C, /* output image */
  SAL_i32 eflag ); /* ESAL flag */

Description

The f5x5x() and f5x5dx() functions filter an M x N image by performing a two-dimensional convolution with a 5 x 5 kernel. These functions pad the perimeter of output image C with a border of zeros of width 2. These functions can be done out-of-place only.

 

For a general filter that specifies the kernel size, see imgfirx(), imgfirdx()
