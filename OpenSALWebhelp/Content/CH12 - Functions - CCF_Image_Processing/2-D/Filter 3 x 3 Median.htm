Filter, 3 x 3 Median

medf3x3x(), medf3x3dx()

Syntax

void medf3x3x(
  SAL_f32 *A, /* input image */
  SAL_i32 m, /* row count for A and C, >= 3 */
  SAL_i32 n, /* column count for A and C, >= 4 and even */
  SAL_f32 *C, /* output image */
  SAL_i32 eflag ); /* ESAL flag */
void medf3x3dx(
  SAL_f64 *A, /* input image */
  SAL_i32 m, /* row count for A and C, >= 3 */
  SAL_i32 n, /* column count for A and C, >= 4 and even */
  SAL_f64 *C, /* output image */
  SAL_i32 eflag ); /* ESAL flag */

Description

The medf3x3x() and medf3x3dx() functions filter an m x n image by outputting the median value of each 3 x 3 submatrix of image A. These functions pad the perimeter of output image C with a border of zeros of width 1. These functions can be done out-of-place only.
