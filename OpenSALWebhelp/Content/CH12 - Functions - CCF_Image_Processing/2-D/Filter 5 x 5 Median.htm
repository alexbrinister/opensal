Filter, 5 x 5 Median

medf5x5x(), medf5x5dx()

Syntax

void medf5x5x(
  SAL_f32 *A, /* input image */
  SAL_i32 m, /* row count for A and C, >= 5 */
  SAL_i32 n, /* column count for A and C, >= 6 and even */
  SAL_f32 *C, /* output image */
  SAL_i32 eflag ); /* ESAL flag */
void medf5x5dx(
  SAL_f64 *A, /* input image */
  SAL_i32 m, /* row count for A and C, >= 5 */
  SAL_i32 n, /* column count for A and C, >= 6 and even */
  SAL_f64 *C, /* output image */
  SAL_i32 eflag ); /* ESAL flag */

Description

The medf5x5x() and medf5x5dx() functions filter an m x n image by outputting the median value of each 5 x 5 submatrix of image A. These functions pad the perimeter of output image C with a border of zeros of width 2. These functions can be done out-of-place only.
