Image Filter, m x n Convolution

imgfirx(), imgfirdx()

Syntax

void imgfirx(
  SAL_f32 *A, /* input matrix */
  SAL_i32 m, /* row count for A */
  SAL_i32 n, /* column count for A, n >= q and even */
  SAL_f32 *B, /* input matrix of coefficients */
  SAL_f32 *C, /* output matrix */
  SAL_i32 p, /* row count for B */
  SAL_i32 q, /* column count for B */
  SAL_i32 eflag ); /* ESAL flag */
void imgfirdx(
  SAL_f64 *A, /* input matrix */
  SAL_i32 m, /* row count for A */
  SAL_i32 n, /* column count for A, n >= q and even */
  SAL_f64 *B, /* input matrix of coefficients */
  SAL_f64 *C, /* output matrix */
  SAL_i32 p, /* row count for B */
  SAL_i32 q, /* column count for B */
  SAL_i32 eflag ); /* ESAL flag */

Description

The imgfirx() and imgfirdx() functions filter an image by performing a two-dimensional convolution with a p x q kernel. The imgfirx() and imgfirdx() functions pad the perimeter of output image C with a border of (p-1)/2 rows of zeros on the top and bottom and (q-1)/2 columns of zeros on the left and right. These functions can perform only out-of-place.

For best performance, specify q ≥p.
