Two-Dimensional Convolution

conv2dx(), cconv2dx(), zconv2dx()

Syntax

void conv2dx(
  SAL_f32 *A, /* input matrix */
  SAL_i32 tca, /* total columns in A; tca >= nca */
  SAL_f32 *K, /* kernel matrix */
  SAL_f32 *C, /* output matrix */
  SAL_i32 tcc, /* total columns in C; tcc >= ncc */
  SAL_i32 dsx, /* desampling factor for each row */
  SAL_i32 dsy, /* desampling factor for each column */
  SAL_i32 nck, /* number of columns in K */
  SAL_i32 nrk, /* number of rows in K */
  SAL_i32 ncc, /* number of columns in C */
  SAL_i32 nrc, /* number of rows in C */
  SAL_i32 cflag /* reserved, specify 0 */
  SAL_i32 eflag ); /* ESAL flag */
void cconv2dx(
  SAL_cf32 *A, /* input matrix */
  SAL_i32 tca, /* total columns in A; tca >= nca */
  SAL_cf32 *K, /* kernel matrix */
  SAL_cf32 *C, /* output matrix */
  SAL_i32 tcc, /* total columns in C; tcc >= ncc */
  SAL_i32 dsx, /* desampling factor for each row */
  SAL_i32 dsy, /* desampling factor for each column */
  SAL_i32 nck, /* number of columns in K */
  SAL_i32 nrk, /* number of rows in K */
  SAL_i32 ncc, /* number of columns in C */
  SAL_i32 nrc, /* number of rows in C */
  SAL_i32 cflag /* conjugation flag */
  SAL_i32 eflag ); /* ESAL flag */
void zconv2dx(
  SAL_zf32 *A, /* input matrix */
  SAL_i32 tca, /* total columns in A; tca >= nca */
  SAL_zf32 *K, /* kernel matrix */
  SAL_zf32 *C, /* output matrix */
  SAL_i32 tcc, /* total columns in C; tcc >= ncc */
  SAL_i32 dsx, /* desampling factor for each row */
  SAL_i32 dsy, /* desampling factor for each column */
  SAL_i32 nck, /* number of columns in K */
  SAL_i32 nrk, /* number of rows in K */
  SAL_i32 ncc, /* number of columns in C */
  SAL_i32 nrc, /* number of rows in C */
  SAL_i32 cflag /* conjugation flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The 2-D convolution functions perform a two-dimensional convolution or correlation on the input matrix, using an nrk x nck kernel. These functions can run in-place.

The arguments must have the following relationships:

nca = (ncc - 1) * dsx + nck

nra = (nrc - 1) * dsy + nrk

where nra is the number of rows in A, and nca is the number of columns in A.

Presently, the conjugation flag, cflag, has no effect in conv2dx() and should always be zero. In cconv2dx() and zconv2dx(), the allowed values for cflag are:

SAL_CONJUGATE

Input matrix A is conjugated as part of the convolution.

0

There is no conjugation.
