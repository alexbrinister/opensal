Filter, 3 x 3 Convolution

f3x3x(), f3x3dx()

Syntax

void f3x3x(
  SAL_f32 *A, /* input image */
  SAL_i32 M, /* row count for A and C */
  SAL_i32 N, /* column count for A and C */
  SAL_f32 *B, /* 3x3 input matrix of coefficients */
  SAL_f32 *C /* output image */
  SAL_i32 eflag ); /* ESAL flag */
void f3x3dx(
  SAL_f64 *A, /* input matrix */
  SAL_i32 M, /* row count for A and C */
  SAL_i32 N, /* column count for A and C */
  SAL_f64 *B, /* 3x3 input matrix of coefficients */
  SAL_f64 *C /* output matrix */
  SAL_i32 eflag ); /* ESAL flag */

Description

The f3x3x() and f3x3dx() functions filter an M x N image by performing a two-dimensional convolution with a 3 x 3 kernel. These functions pad the perimeter of output image C with a border of zeros of width 1. These functions can be done out-of-place only.

 

For a general filter that specifies the kernel size, see imgfirx(), imgfirdx().
