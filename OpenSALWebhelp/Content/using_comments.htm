Using Comments

In an effort to continually improve the customer experience, Mercury Computer Systems invites you to submit comments and rate the information provided with this product.

To use the feedback options, your system must be connected to the Internet.

To submit a topic rating:

  • Click the star rating system in the toolbar above.

Topic comments appear at the bottom of this window. The commenting tools appear in the lower-right corner of this window.

To submit a comment:

  • Click the Add comment icon .

To view new and existing comments:

  • Click the Refresh comments icon .

To reply to comments made by another user:

  • Click the Reply to comment icon .

Notes 

Please note that comments are moderated and only approved comments will appear for other users to view.

Before you can submit a comment for the first time, you must register with the Mercury documentation comment database. Any information that you submit is used for comment tracking only and is not made available to other parties within or outside of Mercury Computer Systems, Inc.
