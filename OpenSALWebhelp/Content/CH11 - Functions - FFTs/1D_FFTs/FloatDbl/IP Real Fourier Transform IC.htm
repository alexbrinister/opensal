In-place Real Fourier Transform, Interleaved Complex Format

fft_ripx(), fft_ripdx(), fft_riptx(), fft_riptdx()

Syntax

void fft_ripx(
  FFT_setup *setup, /* weights array */
  SAL_f32 *C, /* input/output vector */
  SAL_i32 k, /* element stride for C */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_ripdx(
  FFT_setupd *setup, /* weights array */
  SAL_f64 *C, /* input/output vector */
  SAL_i32 k, /* element stride for C */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_riptx(
  FFT_setup *setup, /* weights array */
  SAL_f32 *C, /* input/output vector */
  SAL_i32 k, /* element stride for C */
  SAL_f32 *T, /* temporary vector */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_riptdx(
  FFT_setupd *setup, /* weights array */
  SAL_f64 *C, /* input/output vector */
  SAL_i32 k, /* element stride for C */
  SAL_f64 *T, /* temporary vector */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The fft_ripx(), fft_ripdx(), fft_riptx(), and fft_riptdx() functions compute an in-place real discrete Fourier transform:

  • From the time domain to the frequency domain (forward)

    - or - 

  • From the frequency domain to the time domain (inverse)

Forward transforms read real input and write packed complex output. Inverse transforms read packed complex input and write real output. As a result of packing the frequency-domain data, time-domain data and its equivalent frequency-domain data have the same storage requirements.

Parameters

setup

Points to a structure initialized by a prior call to an FFT weights array function, fft1d_setup(). The value supplied as parameter N of the setup function must equal or exceed the value supplied as parameter N of the transform function.

k

Specifies an address stride through input/output vector C. Specify 1 to process every element, 2 to process every other element, and so on.

T

A temporary vector for storing interim results of fft_riptx() and fft_riptdx(). Must be large enough to hold the element count as specified by parameter N.

log2N

Base 2 exponent of the number of elements to process. For example, to process 1024 elements, specify 10 for parameter log2N.

f

Forward/inverse directional flag. Specify one of the following:

FFT_FORWARD

Forward transform (real to packed complex).

FFT_INVERSE

Inverse transform (packed complex to real).

Results are undefined for other values of f.
