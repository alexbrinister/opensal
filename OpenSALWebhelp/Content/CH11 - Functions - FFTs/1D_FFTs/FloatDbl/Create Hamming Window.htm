Create Hamming Window

hamm_windowx(), hammd_windowx()

Syntax

void hamm_windowx(
  SAL_f32 *C, /* output vector */
  SAL_i32 N, /* real element count, N >= 16 */
  SAL_i32 flag, /* flags; specify SAL_HALF_WINDOW to create only first (N+1)/2 pts */
  SAL_i32 eflag ); /* ESAL flag */
void hammd_windowx(
  SAL_f64 *c, /* output vector */
  SAL_i32 n, /* double-precision element count, N >= 16 */
  SAL_i32 flag, /* flags; specify SAL_HALF_WINDOW to create only first (N+1)/2 pts */
  SAL_i32 eflag ); /* ESAL flag */

Description

The hamm_windowx() and hammd_windowx() functions create a Hamming window function, which can be multiplied by a vector using vmulx() or vmuldx().

To create only the first (N + 1)/2 points, specify the SAL_HALF_WINDOW flag.

Related Functions
