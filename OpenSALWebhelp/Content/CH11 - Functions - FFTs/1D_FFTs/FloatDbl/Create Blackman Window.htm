Create Blackman Window

blkman_windowx(), blkmand_windowx()

Syntax

void blkman_windowx(
  SAL_f32 *c, /* output vector */
  SAL_i32 N, /* real element count */
  SAL_i32 fflag, /* flags */
  SAL_i32 eflag ); /* ESAL flag */
void blkmand_windowx(
  SAL_f64 *c, /* output vector */
  SAL_i32 N, /* double-precision element count */
  SAL_i32 fflag, /* flags */
  SAL_i32 eflag ); /* ESAL flag */

Description

The blkman_windowx() and blkmand_windowx() functions create a Blackman window function, which can be multiplied by a vector using vmulx() or vmuldx().

Specify SAL_HALF_WINDOW for fflag to create only the first (n+1)/2 points, or zero for full size window.

Related Functions 

vmulx(), vmuldx(), vmulix()
