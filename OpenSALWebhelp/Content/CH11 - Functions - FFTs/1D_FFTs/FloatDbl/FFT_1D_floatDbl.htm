One-Dimensional Precision Fast Fourier Transforms, Float and Double

One-dimensional real FFT window float functions perform one-dimensional FFTs and compute associated windows for float data.

One-dimensional real FFT window double precision functions perform one-dimensional FFTs and compute associated windows for double precision data.
