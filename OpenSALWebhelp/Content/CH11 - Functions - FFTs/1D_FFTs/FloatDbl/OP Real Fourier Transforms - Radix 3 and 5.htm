Out-of-place Real Fourier Transforms, Radix 3 and Radix 5

fft3_ropx(), fft3_ropdx(), fft5_ropx(), fft5_ropdx()

Syntax

void fft3_ropx( /* single precision function */
  FFT_setup *setup, /* initialized setup structure */
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* half-element stride for A */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* half-element stride for C */
  SAL_ui32 log2M, /* base 2 log of 1/3 transform size*/
  SAL_i32 f, /* direction, 1 (forward) or -1 (inverse) */
  SAL_i32 eflag ); /* ESAL flag */
void fft3_ropdx( /* double precision function */
  FFT_setupd *setup, /* initialized setup structure */
  SAL_f64 *A, /* input vector */
  SAL_i32 i, /* half-element stride for A */
  SAL_f64 *C, /* output vector */
  SAL_i32 k, /* half-element stride for C */
  SAL_ui32 log2M, /* base 2 log of 1/3 transform size*/
  SAL_i32 f, /* direction, 1 (forward) or -1 (inverse) */
  SAL_i32 eflag ); /* ESAL flag */
void fft5_ropx( /* single precision function */
  FFT_setup *setup, /* initialized setup structure */
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* half-element stride for A */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* half-element stride for C */
  SAL_ui32 log2M, /* base 2 log of 1/5 transform size*/
  SAL_i32 f, /* direction, 1 (forward) or -1 (inverse) */
  SAL_i32 eflag ); /* ESAL flag */
void fft5_ropdx( /* double precision function */
  FFT_setupd *setup, /* initialized setup structure */
  SAL_f64 *A, /* input vector */
  SAL_i32 i, /* half-element stride for A */
  SAL_f64 *C, /* output vector */
  SAL_i32 k, /* half-element stride for C */
  SAL_ui32 log2M, /* base 2 log of 1/5 transform size*/
  SAL_i32 f, /* direction, 1 (forward) or -1 (inverse) */
  SAL_i32 eflag ); /* ESAL flag */

Description

The fft3_ropx() and fft3_ropdx() functions compute an out-of-place radix 3 real Fourier transform, forward or inverse. The number of input and output values processed equals three times the power of 2 specified by parameter log2M.

fft5_ropx() and fft5_ropdx() compute an out-of-place radix 5 real Fourier transform, forward or inverse. The number of input and output values processed equals five times the power of 2 specified by parameter log2M.

Parameters

M

Parameter M must be less than or equal to the N argument to fft1d_setup().

FFT_setup and FFT_setupd

The FFT_setup parameter is initialized using fft1d_setup() for single precision functions, or fft1d_setupd() for double precision functions. The FFT_setup and FFT_setupd parameters are preserved for reuse.

f

Assign directional flag f a value of 1 for forward transforms or –1 for inverse transforms. Results are undefined for other values of f. Do not specify the same buffer for vector A and vector C. Vector A is not preserved.
