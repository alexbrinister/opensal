Create Hanning Window

hann_windowx(), hannd_windowx()

Where:

W= 0.5 for a denormalized window (default)

W=0.8165 for a normalized window

Syntax

void hann_windowx(
  SAL_f32 *C, /* output vector */
  SAL_i32 N, /* real element count */
  SAL_i32 flag, /* window flags */
  SAL_i32 eflag ); /* ESAL flag */
void hannd_windowx(
  SAL_f64 *C, /* output vector */
  SAL_i32 N, /* double-precision element count */
  SAL_i32 flag, /* window flags */
  SAL_i32 eflag ); /* ESAL flag */

Description

The hann_windowx() and hannd_windowx() functions create a Hanning window function, which can be multiplied by a vector using vmulx() or vmuldx().

The window flag parameter can have the following values:

SAL_HANN_DENORM

Creates a denormalized window (default).

SAL_HANN_NORM

Creates a normalized window.

SAL_HALF_WINDOW

Creates only the first (N + 1)/2 points.

Related Functions 

vmulx(), vmuldx(), vmulix()
