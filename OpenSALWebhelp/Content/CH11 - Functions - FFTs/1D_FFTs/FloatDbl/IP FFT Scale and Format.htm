In-Place Fourier Transform Scale and Format

rfftscx(), rfftscdx()

Syntax

void rfftscx(
  SAL_f32 *C, /* input-output vector */
  SAL_i32 n, /* real element count */
  SAL_i32 f, /* format flag */
  SAL_i32 g, /* scaling flag */
  SAL_i32 eflag ); /* ESAL flag */
void rfftscdx(
  SAL_f64 *C, /* input-output vector */
  SAL_i32 n, /* double-precision element count */
  SAL_i32 f, /* format flag */
  SAL_i32 g, /* scaling flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The rfftscx() and rfftscdx() functions scale and format frequency-domain data produced by real discrete Fourier transforms. You can use the rfftscx() or rfftscdx() function to:

  • Unpack frequency-domain data that the forward real DFTs produce
  • Pack complex data for input to inverse real DFTs

The f format flag specifies the following output formats:

0 Leave format unchanged
2 Unpack to N/2 complex elements
-2 Pack N/2 complex elements into rfft() format
3 Unpack to N/2+1 complex elements
-3 Pack N/2+1 complex elements into rfft() format

The g scaling flag specifies output scaling as:

g = 0 No scaling
g > 0 Multiply by 1/2N
g < 0 Multiply by 1/4N
