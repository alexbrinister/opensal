Out-of-place Real Fourier Transforms, Split Complex Format, Radix 3 and Radix 5

fft3_zropx(), fft3_zropdx(), fft5_zropx(), fft5_zropdx()

Syntax

void fft3_zropx(
  FFT_setup *setup, /* initialized setup structure */
  SAL_zf32 *A, /* input vector */
  SAL_i32 i, /* stride for A */
  SAL_zf32 *C, /* output vector */
  SAL_i32 j, /* stride for C */
  SAL_ui32 log2m, /* base 2 log of 1/3 FFT size */
  SAL_i32 f, /* direction flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft3_zropdx(
  FFT_setupd *setup, /* initialized setup structure */
  SAL_zf64 *A, /* input vector */
  SAL_i32 i, /* stride for A */
  SAL_zf64 *C, /* output vector */
  SAL_i32 j, /* stride for C */
  SAL_ui32 log2m, /* base 2 log of 1/3 FFT size */
  SAL_i32 f, /* direction flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft5_zropx(
  FFT_setup *setup, /* initialized setup structure */
  SAL_zf32 *A, /* input vector */
  SAL_i32 i, /* stride for A */
  SAL_zf32 *C, /* output vector */
  SAL_i32 j, /* stride for C */
  SAL_ui32 log2m, /* base 2 log of 1/5 FFT size */
  SAL_i32 f, /* direction flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft5_zropdx(
  FFT_setupd *setup, /* weights array */
  SAL_zf64 *A, /* input vector */
  SAL_i32 i, /* stride for A */
  SAL_zf64 *C, /* output vector */
  SAL_i32 j, /* stride for C */
  SAL_ui32 log2m, /* base 2 log of 1/5 FFT size */
  SAL_i32 f, /* direction flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The fft3_zropx() and fft3_zropdx() functions compute an out-of-place, split complex format, radix 3 real Fourier transform, forward or inverse. The number of input and output values processed equals three times the power of 2 specified by parameter log2m.

The fft5_zropx() and fft5_zropdx() functions compute an out-of-place, split complex format, radix 5 real Fourier transform, forward or inverse. The number of input and output values processed equals five times the power of 2 specified by parameter log2m.

Parameters

FFT_setup and FFT_setupd

For single precision FFT functions, the FFT_setup parameter is initialized using fft1d_setup(). For double precision FFT functions, the FFT_setupd parameter is initialized using fft1d_setupd(). The FFT_setup and FFT_setupd parameters are preserved for reuse.

FFT_REAL_ONLY

For optimal performance:

  • When setting up for fft3_zropx() or fft5_zropx(), pass FFT_REAL_ONLY to the fft1d_setup() flag.
  • When setting up for fft3_zropdx() or fft5_zropdx(), pass FFT_REAL_ONLY to the fft1d_setupd() flag.

f

For forward transforms, assign directional flag f a value of FFT_FORWARD.

For inverse transforms, assign directional flag f a value of FFT_INVERSE.

Results are undefined for other values of f.

A and C

Do not specify the same buffer for vector A and vector C. Vector A is destroyed.
