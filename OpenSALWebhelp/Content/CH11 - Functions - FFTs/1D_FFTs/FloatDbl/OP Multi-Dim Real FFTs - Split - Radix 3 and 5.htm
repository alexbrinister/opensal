Dimensional Multiple Out-of-place Real Fourier Transforms, Split Complex Format, Radix 3 and Radix 5

fft3m_rows_zropx(), fft3m_cols_zropx(), fft5m_rows_zropx(), fft5m_cols_zropx()

Syntax

fft3m_rows_zropx(
  FFT_setup *setup, /* weights array */
  SAL_zf32 *A, /* input component matrices (destroyed) */
  SAL_i32 Atcols, /* column stride for A */
  SAL_zf32 *C, /* output vector */
  SAL_i32 Ctcols, /* column stride for C */
  SAL_i32 log2n, /* base 2 exponent of element count */
  SAL_i32 nffts, /* number of FFTs in the batch*/
  SAL_i32 fi_flag, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The fft3m_rows_zropx() and fft3m_cols_zropx() functions compute multiple out-of-place split complex radix-3 discrete Fourier transforms, for rows or columns only, respectively.

The fft5m_rows_zropx() and fft5m_cols_zropx() functions compute multiple out-of-place split complex radix-5 discrete Fourier transforms, for rows or columns only, respectively.

Parameters

setup

Points to a structure initialized by a prior call to FFT weights function, fft1d_setup().

log2n

The base 2 exponent of the transform size (number of elements). For example, a value of 7 specifies a size of 384 for a radix-3 FFT.

Atcols

Specifies a column stride through input vector A. To process every element, specify a stride of 1. To process every other element, specify a stride of 2.

Ctcols

Specifies a column stride through output vector C. To process every element, specify a stride of 1. To process every other element, specify a stride of 2.

nffts

The number of FFTs in the batch to process.

fi_flag

A forward/inverse directional flag that must specify one of the following:

FFT_FORWARD

Forward transform.

FFT_INVERSE

Inverse transform.

flag

The ESAL flag.
