Dimensional Multiple In-place Real Fourier Transform, Interleaved Complex Format

fftm_rows_ripx(), fftm_cols_ripx(), fftm_rows_ripdx(), fftm_cols_ripdx()

Syntax

fftm_rows_ripx(
  FFT_setup *setup, /* weights array */
  SAL_f32 *C, /* output component matrices */
  SAL_i32 Ctcols, /* column stride for C */
  SAL_i32 log2n, /* base 2 exponent of element count */
  SAL_i32 nffts, /* number of FFTs in the batch*/
  SAL_i32 fi_flag, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
fftm_cols_ripx(
  FFT_setup *setup, /* weights array */
  SAL_f32 *C, /* output component matrices */
  SAL_i32 Ctcols, /* column stride for C */
  SAL_i32 log2n, /* base 2 exponent of element count */
  SAL_i32 nffts, /* number of FFTs in the batch*/
  SAL_i32 fi_flag, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
fftm_rows_ripdx(
  FFT_setupd *setup, /* weights array */
  SAL_f64 *C, /* output component matrices */
  SAL_i32 Ctcols, /* column stride for C */
  SAL_i32 log2n, /* base 2 exponent of element count */
  SAL_i32 nffts, /* number of FFTs in the batch*/
  SAL_i32 fi_flag, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
fftm_cols_ripdx(
  FFT_setupd *setup, /* weights array */
  SAL_f64 *C, /* output component matrices */
  SAL_i32 Ctcols, /* column stride for C */
  SAL_i32 log2n, /* base 2 exponent of element count */
  SAL_i32 nffts, /* number of FFTs in the batch*/
  SAL_i32 fi_flag, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The fftm_rows_rip[d]x() and fftm_cols_rip[d]x() functions compute multiple in-place real discrete Fourier transforms, for rows or columns only, respectively.

These functions perform the same as the fftm_rip[d]x() functions, but operate on row-wise or column-wise data layouts only.

Parameters

setup

Points to a structure initialized by a prior call to fft1d_setup[d](), specifying the FFT function (e.g., SAL_FFTM_ROWS_RIPX)

Ctcols

Specifies a column stride for vector C. For row FFTs, Ctcols should be greater or equal to the number of elements in the FFT. For column FFTs, Ctcols should be greater or equal to nffts.

log2n

The base-2 exponent of the number of complex elements to process. For example, to process 1,024 elements, specify 10 for the log2N parameter. This must be the same value specified to the fft1d_setup[d]() routine.

nffts

The number of FFTs in the batch to process.

fi_flag

A forward/inverse directional flag that must specify one of the following:

FFT_FORWARD

Forward transform.

FFT_INVERSE

Inverse transform.

flag

The ESAL flag.
