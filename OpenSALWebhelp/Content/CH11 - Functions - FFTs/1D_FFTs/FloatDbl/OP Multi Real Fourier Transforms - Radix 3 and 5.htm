Multiple Out-of-place Real Fourier Transforms, Radix 3 and Radix 5

fft3m_ropx(), fft5m_ropx()

Syntax

void fft3m_ropx(
  FFT_setup *setup, /* weights array */
  SAL_f32 *A, /* input matrix (destroyed) */
  SAL_i32 ir, /* adjacent component element stride within each FFT in matrix A */
  SAL_i32 ic, /* corresponding component element stride in adjacent FFTs in matrix A */
  SAL_f32 *C, /* output matrix */
  SAL_i32 kr, /* adjacent component element stride within each FFT in matrix C */
  SAL_i32 kc, /* corresponding component element
  stride in adjacent FFTs in matrix C */
  SAL_ui32 log2n, /* base 2 log of 1/3 FFT size */
  SAL_ui32 m, /* number of FFTs to perform */
  SAL_i32 f, /* forward or inverse FFT */
  SAL_i32 eflag ); /* ESAL flag */
void fft5m_ropx(
  FFT_setup *setup, /* weights array */
  SAL_f32 *A, /* input matrix (destroyed) */
  SAL_i32 ir, /* adjacent component element stride within each FFT in matrix A */
  SAL_i32 ic, /* corresponding component element stride in adjacent FFTs in matrix A */
  SAL_f32 *C, /* output matrix */
  SAL_i32 kr, /* adjacent component element stride within each FFT in matrix C */
  SAL_i32 kc, /* corresponding component element
  stride in adjacent FFTs in matrix C */
  SAL_ui32 log2n, /* base 2 log of 1/5 FFT size */
  SAL_ui32 m, /* number of FFTs to perform */
  SAL_i32 f, /* forward or inverse FFT */
  SAL_i32 eflag ); /* ESAL flag */

Description

The fft3m_ropx() function computes multiple out-of-place real radix 3 discrete Fourier transforms, forward or inverse.

The fft5m_ropx() function computes multiple out-of-place real radix 5 discrete Fourier transforms, forward or inverse.

Parameters

setup

Points to a structure initialized by a prior call to FFT weights function, fft1d_setup().

A and C

Input matrix A and output matrix C have two dimensions. One dimension is m, the number of FFTs to perform. The other dimension is the size of each FFT, equal to rn, where r is the radix.

log2n

The base 2 exponent of the size of the transform (number of elements). For example, a value of 7 specifies a size of 384 for a radix-3 FFT.

ir and kr

Each matrix has two stride parameters.

  • The ir and kr parameters specify the stride between neighboring component elements within each FFT sequence.
  • The ic and kc parameters specify the stride between corresponding component elements in the neighboring FFT sequences.

To prevent multiple FFT sequence overlapping, one of the following conditions must exist:

  • ic must be greater than or equal to r x n x ir

    - or - 

  • ir must be greater than or equal to m x ic

Similarly, one of the following conditions must exist:

  • kc must be greater than or equal to r x n x kr

    - or - 

  • kr must be greater than or equal to m x kc

The A and C matrix parameters must specify different matrices. The contents of A are not preserved.

f

Set to FFT_FORWARD for a forward transform or FFT_INVERSE for an inverse transform.
