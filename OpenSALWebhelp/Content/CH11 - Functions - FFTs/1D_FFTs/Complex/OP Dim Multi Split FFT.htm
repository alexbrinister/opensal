Dimensional Multiple Out-of-Place Split Complex Fourier Transforms

fftm_rows_zopx(), fftm_cols_zopx(), fftm_rows_zopdx(), fftm_cols_zopdx()

Syntax

fftm_rows_zopx (
  FFT_setupd *setup, /* weights array */
  SAL_zf32 *A, /* input vector */
  SAL_i32 Atcols, /* column stride for A */
  SAL_zf32 *C, /* output vector */
  SAL_i32 Ctcols, /* column stride for C */
  SAL_i32 log2n, /* base 2 exponent of element count */
  SAL_i32 nffts, /* number of FFTs in the batch*/
  SAL_i32 fi_flag, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
fftm_cols_zopx (
  FFT_setupd *setup, /* weights array */
  SAL_zf32 *A, /* input vector */
  SAL_i32 Atcols, /* column stride for A */
  SAL_zf32 *C, /* output vector */
  SAL_i32 Ctcols, /* column stride for C */
  SAL_i32 log2n, /* base 2 exponent of element count */
  SAL_i32 nffts, /* number of FFTs in the batch*/
  SAL_i32 fi_flag, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
fftm_rows_zopdx (
  FFT_setupd *setup, /* weights array */

  SAL_i32 Atcols, /* column stride for A */
  SAL_zf64 *C, /* output vector */
  SAL_i32 Ctcols, /* column stride for C */

  SAL_i32 eflag ); /* ESAL flag */

Description

The fftm_rows_zop[d]x() and fftm_cols_zop[d]x() functions compute multiple out-of-place split complex discrete Fourier transforms, for rows or columns only, respectively.

These functions perform the same as the fftm_zop[d]x() functions, but operate on row-wise or column-wise data layouts only.

Parameters

setup

Points to a structure initialized by a prior call to fft1d_setup[d](), specifying the FFT function (e.g., SAL_FFTM_ROWS_ZIPX)

Atcols

Specifies a column stride for vector A. For row FFTs, Atcols should be greater or equal to the number of elements in the FFT. For column FFTs, Atcols should be greater or equal to nffts.

Ctcols

Specifies a column stride for vector C. For row FFTs, Ctcols should be greater or equal to the number of elements in the FFT. For column FFTs, Ctcols should be greater or equal to nffts.

log2n

The base-2 exponent of the number of complex elements to process. For example, to process 1,024 elements, specify 10 for the log2N parameter. This must be the same value specified to the fft1d_setup[d]() routine.

nffts

The number of FFTs in the batch to process.

fi_flag

A forward/inverse directional flag that must specify one of the following:

FFT_FORWARD

Forward transform.

FFT_INVERSE

Inverse transform.

flag

The ESAL flag.
