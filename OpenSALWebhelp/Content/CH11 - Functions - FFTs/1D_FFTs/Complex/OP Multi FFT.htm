Multiple Out-of-Place Complex Fourier Transform

fftm_copx(), fftm_copdx(), fftm_coptx(), fftm_coptdx()

Syntax

void fftm_copx(
  FFT_setup *setup, /* weights array */
  SAL_cf32 *A, /* input vector (destroyed) */
  SAL_i32 i, /* address stride for A, must be even */
  SAL_i32 j, /* stride between FFTs */
  SAL_cf32 *C, /* output vector */
  SAL_i32 k, /* address stride for C, must be even */
  SAL_i32 l, /* stride between FFTs */
  SAL_ui32 log2n, /* base 2 exponent of element count */
  SAL_ui32 m, /* number of FFTs in the batch */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fftm_copdx(
  FFT_setupd *setup, /* weights array */
  SAL_cf64 *A, /* input vector (destroyed) */
  SAL_i32 i, /* address stride for A, must be even */
  SAL_i32 j, /* stride between FFTs */
  SAL_cf64 *C, /* output vector */
  SAL_i32 k, /* address stride for C, must be even */
  SAL_i32 l, /* stride between FFTs */
  SAL_ui32 log2n, /* base 2 exponent of element count */
  SAL_ui32 m, /* number of FFTs in the batch */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fftm_coptx(
  FFT_setupd *setup, /* weights array */
  SAL_cf32 *A, /* input vector */
  SAL_i32 i, /* address stride for A, must be even */
  SAL_i32 j, /* stride between FFTs */
  SAL_cf32 *C, /* output vector */
  SAL_i32 k, /* address stride for C, must be even */
  SAL_i32 l, /* stride between FFTs */
  SAL_cf32 *T, /* temporary vector */
  SAL_ui32 log2n, /* base 2 exponent of element count */
  SAL_ui32 m, /* number of FFTs in the batch */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fftm_coptdx(
  FFT_setupd *setup, /* weights array */
  SAL_cf64 *A, /* input vector */
  SAL_i32 i, /* address stride for A, must be even */
  SAL_i32 j, /* stride between FFTs */
  SAL_cf64 *C, /* output vector */
  SAL_i32 k, /* address stride for C, must be even */
  SAL_i32 l, /* stride between FFTs */
  SAL_cf64 *T, /* temporary vector */
  SAL_ui32 log2n, /* base 2 exponent of element count */
  SAL_ui32 m, /* number of FFTs in the batch */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

These functions are multiple (batch) versions of the fft_copx(), fft_coptx(), fft_copdx(), and fft_coptdx() functions. Use these functions to efficiently process small FFTs (less than 512 points). These functions work for any FFT of 4 points or larger.
