In-place Fourier Transform Scale and Format

cfftscx(), cfftscdx()

Syntax

void cfftscx(
  SAL_cf32 *C, /* input-output vector */
  SAL_i32 N, /* complex element count */
  SAL_i32 eflag ); /* ESAL flag */
void cfftscdx(
  SAL_cf64 *C, /* input-output vector */
  SAL_i32 N, /* complex element count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The cfftscx() and cfftscdx() functions scale the results of inverse complex-to-complex discrete Fourier transforms. Vector C is scaled in-place by 1/N.
