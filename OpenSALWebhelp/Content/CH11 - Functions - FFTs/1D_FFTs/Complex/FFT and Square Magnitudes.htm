Fourier Transform and Square Magnitudes

fft_cvmagsx(), fft_zvmagsx()

Syntax

void fft_cvmagsx(
  FFT_setup *setup, /* weights array */
  SAL_cf32 *A, /* input vector (destroyed) */
  SAL_i32 i, /* half-element stride for A, must be even */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* element stride for C */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 flag, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_zvmagsx(
  FFT_setup *setup, /* weights array */
  SAL_zf32 *A, /* input vector (destroyed) */
  SAL_i32 i, /* element stride for A */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* element stride for C */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 flag, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The fft_cvmagsx() and fft_zvmagsx() functions calculate the in-place complex FFT and squared magnitudes of complex vector A.

  • A call to fft_cvmagsx() produces the same output as calling fft_cipx() followed by cvmagsx().
  • A call to fft_zvmagsx() produces the same output as a pair of calls, fft_zipx() followed by zvmagsx().

In both cases, the output from the first call is supplied as input to the second call.

Both functions destroy the complex input vector, but they can both be in-place operations:

  • For the fft_cvmagsx(), if both vectors are packed (i = 2 and k = 1), the output vector C can be the same as the input vector A. Also, if the two strides, i and k, have the same value, C can point to the real (&A->real) or imaginary (&A->imag) input vector component.
  • For the fft_zvmagsx(), if i and k have the same value, the output vector C can point to the real (A->realp) or imaginary (A->imagp) input vector component.

Parameters

setup

Points to a structure initialized by a prior call to FFT weights array function, fft1d_setup().

i, k

Element strides through input vector A and output vector C. For the fft_cvmagsx() function, i is a half-element stride and must be even.

log2N

Base 2 exponent of the number of elements to process. For example, to process 1024 elements, specify 10 for parameter log2N.

flag

Forward/inverse directional flag. Specify one of the following:

FFT_FORWARD

Forward transform.

FFT_INVERSE

Inverse transform.

Results are undefined for other values of flag.
