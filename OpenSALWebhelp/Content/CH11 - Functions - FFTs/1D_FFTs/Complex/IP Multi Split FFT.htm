fftm_zipx(), fftm_zipdx(), fftm_ziptx(), fftm_ziptdx()

  SAL_zf32 *C, /* input/output vector */

  SAL_i32 l, /* stride between FFTs */

Description

The fftm_zipx(), fftm_zipdx(), fftm_ziptx(), and fftm_ziptdx() functions are multiple (batch) versions of the fft_zipx(), fft_ziptx(), fft_zipdx(), and fft_ziptdx() functions. Use these functions to efficiently process small FFTs (less than 512 points). These functions work for any FFT of 4 points or larger.
