Dimensional Multiple Out-of-place Complex Fourier Transforms, Radix 3 and 5

fft3m_rows_copx(), fft3m_cols_copx(), fft5m_rows_copx(), fft5m_cols_copx()

Syntax

fft3m_rows_copx(
  FFT_setup *setup, /* weights array */
  SAL_cf32 *A, /* input component matrices (destroyed) */
  SAL_i32 Atcols, /* column stride for A */
  SAL_cf32 *C, /* output vector */
  SAL_i32 Ctcols, /* column stride for C */
  SAL_i32 log2n, /* base 2 exponent of element count */
  SAL_i32 nffts, /* number of FFTs in the batch*/
  SAL_i32 fi_flag, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
fft3m_cols_copx(
  FFT_setup *setup, /* weights array */
  SAL_cf32 *A, /* input component matrices (destroyed) */
  SAL_i32 Atcols, /* column stride for A */
  SAL_cf32 *C, /* output vector */
  SAL_i32 Ctcols, /* column stride for C */
  SAL_i32 log2n, /* base 2 exponent of element count */
  SAL_i32 nffts, /* number of FFTs in the batch*/
  SAL_i32 fi_flag, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
fft5m_rows_copx(
  FFT_setup *setup, /* weights array */

  SAL_i32 Ctcols, /* column stride for C */

  SAL_i32 Ctcols, /* column stride for C */

  SAL_i32 eflag ); /* ESAL flag */

Description

The fft3m_rows_copx() and fft3m_cols_copx() functions compute multiple out-of-place complex radix-3 discrete Fourier transforms, for rows or columns only, respectively.

The fft5m_rows_copx() and fft5m_cols_copx() functions compute multiple out-of-place complex radix-5 discrete Fourier transforms, for rows or columns only, respectively.

Parameters

setup

Points to a structure initialized by a prior call to fft1d_setup[d](), specifying the FFT function (e.g., SAL_FFT3M_ROWS_COPX).

Atcols

Specifies a column stride for input vector A. For row FFTs, Atcols should be greater than or equal to the number of elements in the FFT. For column FFTs, Atcols should be greater than or equal to nffts.

Ctcols

Specifies a column stride for output vector C. For row FFTs, Ctcols should be greater than or equal to the number of elements in the FFT. For column FFTs, Ctcols should be greater than or equal to nffts.

log2n

The base-2 exponent of the transform size (number of complex elements). For example, a value of 7 specifies a size of 384 for a radix-3 FFT. This must be the same value specified to the fft1d_setup[d]() routine.

nffts

The number of FFTs in the batch to process.

fi_flag

A forward/inverse directional flag that must specify one of the following:

FFT_FORWARD

Forward transform.

FFT_INVERSE

Inverse transform.

flag

The ESAL flag.
