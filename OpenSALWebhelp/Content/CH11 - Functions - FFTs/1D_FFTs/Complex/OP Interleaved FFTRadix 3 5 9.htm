Out-of-place Interleaved Complex Fourier Transforms, Radix 3, Radix 5 and Radix 9

fft3_copx(), fft3_copdx(), fft5_copx(), fft5_copdx(), fft9_copx()

N = 3(2m) for radix-3 functions

N = 5(2m) for radix-5 functions

N = 9(2m) for radix-9 functions

Syntax

void fft3_copx(
  FFT_setup *setup, /* initialized setup structure */
  SAL_cf32 *A, /* input vector */
  SAL_i32 i, /* half-element stride for A */
  SAL_cf32 *C, /* output vector */
  SAL_i32 k, /* half-element stride for C */
  SAL_ui32 log2M, /* base 2 log of 1/3 transform size*/
  SAL_i32 f, /* direction, 1 (forward) or -1 (inverse) */
  SAL_i32 eflag ); /* ESAL flag */
void fft3_copdx(
  FFT_setupd *setup, /* initialized setup structure */
  SAL_cf64 *A, /* input vector */
  SAL_i32 i, /* half-element stride for A */
  SAL_cf64 *C, /* output vector */
  SAL_i32 k, /* half-element stride for C */
  SAL_ui32 log2M, /* base 2 log of 1/3 transform size*/
  SAL_i32 f, /* direction, 1 (forward) or -1 (inverse) */
  SAL_i32 eflag ); /* ESAL flag */
void fft5_copx(
  FFT_setup *setup, /* initialized setup structure */

  SAL_i32 i, /* half-element stride for A */
  SAL_cf32 *C, /* output vector */
  SAL_i32 k, /* half-element stride for C */
  SAL_ui32 log2m, /* base 2 log of 1/5 transform size*/

  SAL_i32 eflag ); /* ESAL flag */

  SAL_i32 f, /* direction flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The fft3_copx() and fft3_copdx() functions compute an out-of-place radix 3 complex Fourier transform, forward or inverse. The number of input and output values processed equals three times the power of 2 specified by parameter log2M.

The fft5_copx() and fft5_copdx() functions compute an out-of-place radix 5 complex Fourier transform, forward or inverse. The number of input and output values processed equals five times the power of 2 specified by parameter log2M.

The fft9_copx() function computes an out-of-place radix 9 complex Fourier transform, forward or inverse. The number of input and output values processed equals nine times the power of 2 specified by parameter log2M.

Parameters

FFT_setup and FFT_setupd

For single precision functions, FFT_setup is initialized by fft1d_setup(). For double precision functions, FFT_setupd is initialized by fft1d_setupd().

FFT_setup and FFT_setupd are preserved for reuse.

f

Directional flag. Assign f a value of FFT_FORWARD for forward transforms or FFT_INVERSE for inverse transforms. Results are undefined for other values of f.

A and C

Do not specify the same buffer for vector A and vector C. Vector A is not preserved.

M

Parameter M must be less than or equal to the N argument to fft1d_setup().
