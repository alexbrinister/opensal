Out-of-place Split Complex Fourier Transform

fft_zopx(), fft_zopdx(), fft_zoptx(), fft_zoptdx()

Syntax

void fft_zopx(
  FFT_setup *setup, /* weights array */
  SAL_zf32 *A, /* input vector (destroyed) */
  SAL_i32 i, /* element stride for A */
  SAL_zf32 *C, /* output vector */
  SAL_i32 k, /* element stride for C */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_zopdx(
  FFT_setupd *setup, /* weights array */
  SAL_zf64 *A, /* input vector (destroyed) */
  SAL_i32 i, /* element stride for A */
  SAL_zf64 *C, /* output vector */
  SAL_i32 k, /* element stride for C */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_zoptx(
  FFT_setup *setup, /* weights array */
  SAL_zf32 *A, /* input vector (preserved) */
  SAL_i32 i, /* element stride for A */
  SAL_zf32 *C, /* output vector */
  SAL_i32 k, /* element stride for C */
  SAL_zf32 *T, /* temporary vector */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_zoptdx(
  FFT_setupd *setup, /* weights array */
  SAL_zf64 *A, /* input vector (preserved) */
  SAL_i32 i, /* element stride for A */
  SAL_zf64 *C, /* output vector */
  SAL_i32 k, /* element stride for C */
  SAL_zf64 *T, /* temporary vector */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The fft_zopx(), fft_zopdx(), fft_zoptx(), and fft_zoptdx() functions compute an out-of-place complex discrete Fourier transform of vector A, from the time domain to the frequency domain (forward) or from the frequency domain to the time domain (inverse). The fft_zopx() and fft_zopdx() functions destroy the input vector, while fft_zoptx() and fft_zoptdx() preserve it.

Parameters

setup

Points to a structure initialized by a prior call to FFT weights array function fft1d_setup(). The value supplied as parameter N of the setup function must equal or exceed the value supplied as parameter N of the transform function.

i

Specifies an address stride through input vector A. Parameter k specifies an address stride through output vector C. So, to process every element, specify a stride of 1. To process every other element, specify 2.

T

For fft_zoptx() and fft_zoptdx(), the T parameter is a temporary vector used for storing interim results. Vector T must be large enough to hold the element count as specified by parameter N.

log2N

The base 2 exponent of the number of elements to process. For example, to process 1024 elements, specify 10 for parameter log2N.

f

A forward/inverse directional flag, and must specify one of the following:

FFT_FORWARD

Forward transform.

FFT_INVERSE

Inverse transform.

Results are undefined for other values of f.
