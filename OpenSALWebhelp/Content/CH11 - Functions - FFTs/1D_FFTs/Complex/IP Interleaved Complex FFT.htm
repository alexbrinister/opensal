In-place Interleaved Complex Fourier Transform

fft_cipx(), fft_cipdx(), fft_ciptx(), fft_ciptdx()

Syntax

void fft_cipx(
  FFT_setup *setup, /* weights array */
  SAL_cf32 *C, /* input/output vector */
  SAL_i32 k, /* element stride for C, must be even */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_cipdx(
  FFT_setupd *setup, /* weights array */
  SAL_cf64 *C, /* input/output vector */
  SAL_i32 k, /* element stride for C, must be even */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_ciptx(
  FFT_setup *setup, /* weights array */
  SAL_cf32 *C, /* input/output vector */
  SAL_i32 k, /* element stride for C, must be even */
  SAL_cf32 *T, /* temporary vector */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_ciptdx(
  FFT_setupd *setup, /* weights array */
  SAL_cf64 *C, /* input/output vector */
  SAL_i32 k, /* element stride for C, must be even */
  SAL_cf64 *T, /* temporary vector */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The fft_cipx(), fft_cipdx(), fft_ciptx(), and fft_ciptdx() functions compute an in-place complex discrete Fourier transform of vector C, from:

  • The time domain to the frequency domain (forward)

    - or - 

  • The frequency domain to the time domain (inverse)

Parameters

setup

Points to a structure initialized by a prior call to FFT weights array function fft1d_setup(). The setup function n argument must equal to or exceed the transform function n argument.

k

Specifies an address stride through input/output vector C. Because vector C holds complex elements, k must be a multiple of 2. Therefore, to process every element of C, specify 2 for parameter k. To process every other element, specify 4.

T

A temporary vector for storing interim results of fft_ciptx() and fft_ciptdx(). Must be large enough to hold the element count as specified by parameter N.

log2N

Base 2 exponent of the number of elements to process. For example, to process 1024 elements, specify 10 for parameter log2N.

f

A forward/inverse directional flag, and must specify one of the following:

FFT_FORWARD

Forward transform.

FFT_INVERSE

Inverse transform.

Results are undefined for other values of f.
