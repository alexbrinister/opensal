Out-of-place Split Complex Fourier Transforms, Radix 3, Radix 5, and Radix 9

fft3_zopx(), fft3_zopdx(), fft5_zopx(), fft5_zopdx(), fft9_zopx()

N = 3(2m) for radix-3 functions

N = 5(2m) for radix-5 functions

N = 9(2m) for radix-9 functions

Syntax

void fft3_zopx( /* single precision function */
  FFT_setup *setup, /* initialized setup structure */
  SAL_zf32 *A, /* input vector */
  SAL_i32 i, /* stride for A */
  SAL_zf32 *C, /* output vector */
  SAL_i32 j, /* stride for C */
  SAL_ui32 log2M, /* base 2 log of 1/3 transform size*/
  SAL_i32 f, /* direction, 1 (forward) or -1
  (inverse) */  
  SAL_i32 eflag ); /* ESAL flag */
void fft3_zopdx( /* double precision function */
  FFT_setupd *setup, /* initialized setup structure */
  SAL_zf64 *A, /* input vector */
  SAL_i32 i, /* stride for A */
  SAL_zf64 *C, /* output vector */
  SAL_i32 j, /* stride for C */
  SAL_ui32 log2M, /* base 2 log of 1/3 transform size*/
  SAL_i32 f, /* direction, 1 (forward) or -1
  (inverse) */  
  SAL_i32 eflag ); /* ESAL flag */
void fft5_zopx( /* single precision function */
  FFT_setup *setup, /* initialized setup structure */
  SAL_zf32 *A, /* input vector */
  SAL_i32 i, /* stride for A */
  SAL_zf32 *C, /* output vector */
  SAL_i32 j, /* stride for C */
  SAL_ui32 log2M, /* base 2 log of 1/5 transform size*/
  SAL_i32 f, /* direction, 1 (forward) or -1
  (inverse) */  
  SAL_i32 eflag ); /* ESAL flag */
void fft5_zopdx( /* double precision function */
  FFT_setupd *setup, /* initialized setup structure */
  SAL_zf64 *A, /* input vector */
  SAL_i32 i, /* stride for A */
  SAL_zf64 *C, /* output vector */
  SAL_i32 j, /* stride for C */
  SAL_ui32 log2M, /* base 2 log of 1/5 transform size*/
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft9_zopx( /* single precision function */
  FFT_setup *setup, /* initialized setup structure */
  SAL_zf32 *A, /* input vector */
  SAL_i32 i, /* stride for A */
  SAL_zf32 *C, /* output vector */
  SAL_i32 j, /* stride for C */
  SAL_ui32 log2m, /* base 2 log of 1/9 transform size*/
  SAL_i32 f, /* direction flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The fft3_zopx() and fft3_zopdx() functions compute an out-of-place radix 3 complex Fourier transform, forward or inverse. The number of input and output values processed equals three times the power of 2 specified by parameter log2M.

The fft5_zopx() and fft5_zopdx() functions compute an out-of-place radix 5 complex Fourier transform, forward or inverse. The number of input and output values processed equals five times the power of 2 specified by parameter log2M.

The fft9_zopx() function computes an out-of-place radix 9 complex Fourier transform, forward or inverse. The number of input and output values processed equals nine times the power of 2 specified by parameter log2M.

Parameters

FFT_setup and FFT_setupd

For single precision functions, FFT_setup is initialized by fft1d_setup(). For double precision functions, FFT_setupd is initialized by fft1d_setupd().

FFT_setup is preserved for reuse.

f

Assign directional flag f a value of FFT_FORWARD or FFT_INVERSE. Results are undefined for other values of f.

A and C

Do not specify the same buffer for vector A and vector C. Vector A is not preserved.

M

Parameter M must be less than or equal to the N argument to fft1d_setup().
