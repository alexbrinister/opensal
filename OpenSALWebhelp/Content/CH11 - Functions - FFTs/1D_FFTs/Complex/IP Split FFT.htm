In-Place Split Complex Fourier Transform

fft_zipx(), fft_zipdx(), fft_ziptx(), fft_ziptdx()

Syntax

void fft_zipx(
  FFT_setup *setup, /* weights array */
  SAL_zf32 *C, /* input/output vector */
  SAL_i32 k, /* element stride for C */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_zipdx(
  FFT_setupd *setup, /* weights array */
  SAL_zf64 *C, /* input/output vector */
  SAL_i32 k, /* element stride for C */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_ziptx(
  FFT_setup *setup, /* weights array */
  SAL_zf32 *C, /* input/output vector */
  SAL_i32 k, /* element stride for C */
  SAL_zf32 *T, /* temporary vector */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_ziptdx(
  FFT_setupd *setup, /* weights array */
  SAL_zf64 *C, /* input/output vector */
  SAL_i32 k, /* element stride for C */
  SAL_zf64 *T, /* temporary vector */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The fft_zipx(), fft_zipdx(), fft_ziptx(), and fft_ziptdx() functions compute an in-place complex discrete Fourier transform of vector C, from the time domain to the frequency domain (forward) or from the frequency domain to the time domain (inverse).

Parameters

setup

Points to a structure initialized by a prior call to FFT weights array function fft1d_setup(). The value supplied as parameter N of the setup function must equal or exceed the value supplied as parameter N of the transform function.

k

Specifies an address stride through input/output vector C. To process every element of C, specify 1 for parameter k. To process every other element, specify 2.

T

For fft_ziptx() and fft_ziptdx(), the T parameter is a temporary vector used for storing interim results. Vector T must be large enough to hold the element count, as specified by parameter N.

log2N

The base 2 exponent of the number of elements to process. For example, to process 1024 elements, specify 10 for parameter log2N.

f

A forward/inverse directional flag that must specify one of the following:

FFT_FORWARD

Forward transform.

FFT_INVERSE

Inverse transform.

Results are undefined for other values of f.
