Multiple In-Place Interleaved Complex Fourier Transform

fftm_cipx(), fftm_cipdx(), fftm_ciptx(), fftm_ciptdx()

Syntax

void fftm_cipx(
  FFT_setup *setup, /* weights array */
  SAL_cf32 *C, /* input/output vector */
  SAL_i32 k, /* address stride for C, must be even */
  SAL_i32 l, /* stride between FFTs */
  SAL_ui32 log2n, /* base 2 exponent of the element count */
  SAL_ui32 m, /* number of FFTs in the batch */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */

  SAL_ui32 log2n, /* base 2 exponent of element count */

  SAL_ui32 log2n, /* base 2 exponent of element count */

Description

The fftm_cipx(), fftm_cipdx(), fftm_ciptx(), and fftm_ciptdx() functions are multiple (batch) versions of the fft_cipx(), fft_ciptx(), fft_cipdx(), and fft_ciptdx() functions. Use these functions to efficiently process small FFTs (less than 512 points). These functions work for any FFT of 4 points or larger.
