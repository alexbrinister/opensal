Out-of-place Interleaved Complex Fourier Transform

fft_copx(), fft_copdx(), fft_coptx(), fft_coptdx()

Syntax

void fft_copx(
  FFT_setup *setup, /* weights array */
  SAL_cf32 *A, /* input vector (destroyed) */
  SAL_i32 I, /* element stride for A, I=2i */
  SAL_cf32 *C, /* output vector */
  SAL_i32 K, /* element stride for C, K=2k */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_copdx(
  FFT_setupd *setup, /* weights array */
  SAL_cf64 *A, /* input vector (destroyed) */
  SAL_i32 I, /* element stride for A, I=2i */
  SAL_cf64 *C, /* output vector */
  SAL_i32 K, /* element stride for C, K=2k */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_coptx(
  FFT_setup *setup, /* weights array */
  SAL_cf32 *A, /* input vector (preserved) */
  SAL_i32 I, /* element stride for A, I=2i */
  SAL_cf32 *C, /* output vector */
  SAL_i32 K, /* element stride for C, K=2k */
  SAL_cf32 *T, /* temporary vector */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */
void fft_coptdx(
  FFT_setupd *setup, /* weights array */
  SAL_cf64 *A, /* input vector (preserved) */
  SAL_i32 I, /* element stride for A, I=2i */
  SAL_cf64 *C, /* output vector */
  SAL_i32 K, /* element stride for C, K=2k */
  SAL_cf64 *T, /* temporary vector */
  SAL_ui32 log2N, /* base 2 exponent of element count */
  SAL_i32 f, /* directional flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The fft_copx(), fft_copdx(), fft_coptx(), and fft_coptdx() functions compute an out-of-place complex discrete Fourier transform of vector A, from:

  • The time domain to the frequency domain (forward)

    - or - 

  • The frequency domain to the time domain (inverse)

The fft_copx() and fft_copdx() functions destroy the input vector, while fft_coptx() and fft_coptdx() preserve the input vector.

Parameters

setup

Points to a structure initialized by a prior call to FFT weights array function, fft1d_setup(). The value supplied as parameter N of the setup function must equal or exceed the value supplied as parameter N of the transform function.

I, K

Address strides through input vector A and output vector C. Because both vectors hold complex elements, I and K must be multiples of 2. Therefore, to process every element, specify a stride of 2. To process every other element, specify 4.

T

A temporary vector for storing interim results of fft_coptx() and fft_coptdx(). Must be large enough to hold the element count as specified by parameter N.

log2N

Base 2 exponent of the number of elements to process. For example, to process 1024 elements, specify 10 for parameter log2N.

f

Forward/inverse directional flag. Specify one of the following:

FFT_FORWARD

Forward transform.

FFT_INVERSE

Inverse transform.

Results are undefined for other values of f.
