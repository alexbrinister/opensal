FFT Setup Functions

The FFT setup functions acquire and configure the resources required to perform FFT calculations, including a complex exponentials table (a weights array) for use by FFT and windowing functions.

Table 10-1: FFT Setup Functions

FFT FFT Double Description
fft_free fft_freed Frees complex exponentials (weights arrays) created by fft1d_setup() or fft2d_setup() for Fourier transforms
fft1d_setup fft1d_setupd Creates complex exponentials to perform one-dimensional FFT calculations
fft2d_setup fft2d_setupd Creates complex exponentials to perform two-dimensional FFT calculations
fft1d_setup_create fft1d_setupd_create Creates an un-initialized one-dimensional FFT setup handle
fft2d_setup_create fft2d_setupd_create Creates an un-initialized two-dimensional FFT setup handle
fft_setup_init fft_setupd_init Initializes a previously created FFT setup handle
fft_setup_get_n_bytes fft_setupd_get_n_bytes Calculates the number of bytes required to hold FFT setup tables
fft_setup_get_temp_n_bytes fft_setupd_get_temp_n_bytes Gets the number of bytes required by an FFT temporary buffer
fft_setup_set_buffer fft_setupd_set_buffer Provides a buffer to hold the FFT setup handle components

Note 
The fft_setup[d] function has been deprecated. For more information, see Create Complex Exponentials for Fourier Transforms
