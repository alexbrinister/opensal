Create Complex Exponentials for Two-Dimensional Fourier Transforms

fft2d_setup(), fft2d_setupd()

Syntax

SAL_i32 fft2d_setup (
  SAL_fft2d_func fft_type,
  SAL_i32 log2nc,
  SAL_i32 log2nr,
  SAL_i32 flag,
  FFT_setup *setup );
SAL_i32 fft2d_setupd (
  SAL_fft2d_funcd fft_type,
  SAL_i32 log2nc,
  SAL_i32 log2nr,
  SAL_i32 flag,
  FFT_setupd *setup );

Description

The fft2d_setup() and fft2d_setupd() functions acquire and configure the resources required to perform two-dimensional FFT calculations, including tables of complex exponentials (weights arrays) for use by a specific FFT routine.

The fft2d_setup() and fft2d_setupd() routines use an enumerated type argument, whose enumerators represent the individual inner loop routines. This argument refines and optimally configures the resources. Therefore, memory is not allotted to weights arrays, index arrays, and temporary buffers that might never be used.

Parameters

fft_type input

The FFT for which to perform the setup.

log2nr, log2nc input

The base-2 exponent of the number of rows and the number of columns, respectively, in the 2-D FFT matrix.

The setup function uses these size arguments to allocate storage for weights and index arrays, and for any temporary buffers required by the implementation. It also populates these arrays with weights and index values required by the specified FFT routine.

The log2nr and log2nc values specified to the setup routine must also be specified when invoking the actual FFT routine.

p_setup output

A setup handle that points to an opaque structure that is allocated and populated by the setup routine and contains (among other things) pointers to weights and index arrays.

The underlying structure is completely filled in by the function, and should be, subsequently, passed as an argument to the same FFT routine specified in the fft_type parameter.

flag

Reserved; specify SAL_RESERVED_FLAG.

Return Codes

SAL_SUCCESS

The function executed without errors.

SAL_INVALID_FUNCTION

An invalid FFT function type was passed to the function.

SAL_FFT_NOT_SUPPORTED

An invalid enumerator was passed to the function.

SAL_INVALID_SIZE

An invalid element count was passed to the function.

SAL_INVALID_FLAG

An invalid flag was passed to the function.

SAL_INVALID_HANDLE

An invalid or null handle pointer was passed to the function.

SAL_INVALID_ARGUMENT

An invalid argument was passed to the function.

SAL_NOT_ENOUGH_MEMORY

Memory was insufficient for the function to perform all of its operations.

SAL_FAILURE

Function execution failed due to an unknown/uncategorized error.
