Free an Existing Weights Array

fft_free(), fft_freed()

Syntax

void fft_free(
  FFT_setup *setup ); /* specifications */
void fft_freed(
  FFT_setupd *setup ); /* specifications */

Description

The fft_free() and fft_freed() functions free an existing FFT setup handle. Any memory allocated under the handle is released. After the function returns, the handle is no longer valid and cannot be passed to any subsequent FFT or fast convolution functions. To avoid memory leaks, always free the setup handle after processing.

Parameters

setup

An opaque handle that was previously allocated and initialized by one of the following functions:

  • fft1_setup()
  • fft1_setupd()
  • fft2_setup()
  • fft2d_setupd()
