Initialize Buffer for FFT Setup Handle Components

fft_setup_set_buffer(), fft_setupd_set_buffer()

Syntax

int fft_setup_set_buffer (

FFT_setup *p_setup,

void *buffer,

const int nbytes,

const int flag

);

int fft_setupd_set_buffer (

FFT_setup *p_setup,

void *buffer,

const int n_bytes,

const int flag

);

Description

The fft_setup_set_buffer()and fft_setupd_set_buffer() functions provide a buffer to hold the components of an FFT setup handle. These functions allow your application to provide an area of memory to hold the FFT setup components that are necessary to perform FFT calculations, including tables of complex exponentials (weights arrays) for use by the specific FFT routine that was specified when the setup handle was created.

Parameters

p_setup modified

Previously created FFT setup handle.

buffer input

Pointer to the memory area to be used when FFT setup handle is initialized.

n_bytes input

Number of bytes available in the buffer.

flag input

Reserved. Instead, pass SAL_RESERVED_FLAG.

Return Codes

SAL_SUCCESS

The function executed without errors.

SAL_INVALID_SIZE

An invalid buffer size.

SAL_INVALID_FLAG

An invalid flag was passed to the function.

SAL_INVALID_HANDLE

An invalid or null handle pointer was passed to the function.

SAL_INVALID_ARGUMENT

An invalid argument was passed to the function.

SAL_FAILURE

Function execution failed due to an unknown/uncategorized error.
