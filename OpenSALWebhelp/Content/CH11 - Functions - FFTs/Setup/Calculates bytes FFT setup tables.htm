Calculate Number of Bytes to Hold FFT Setup Tables

fft_setup_get_n_bytes(), fft_setupd_get_n_bytes()

Syntax

int fft_setup_get_n_bytes (
FFT_setup *setup,
int *p_nbytes,
const int flag
);
int fft_setupd_get_n_bytes (
FFT_setupd *setup,
int *p_nbytes,
const int flag
);

Description

The fft_setup_get_n_bytes() and fft_setupd_get_n_bytes() calculates the number of bytes required to hold the FFT setup tables. These functions determine the number of bytes necessary to perform FFT calculations, including tables of complex exponentials (weights arrays) for use by the specific FFT routine that was specified when the setup handle was created.

Parameters

setup input

The previously created FFT setup handle.

p_nbytes

The number of bytes required to complete the FFT setup process.

flag input

Reserved. Instead, pass SAL_RESERVED_FLAG.

Return Codes

SAL_SUCCESS

The function executed without errors.

SAL_INVALID_FLAG

An invalid flag was passed to the function.

SAL_INVALID_HANDLE

An invalid or null handle pointer was passed to the function.

SAL_INVALID_ARGUMENT

An invalid argument was passed to the function.

SAL_FAILURE

Function execution failed due to an unknown/uncategorized error.
