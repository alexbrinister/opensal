Two-Dimensional Fast Fourier Transforms, Float and Double Precision

Two-dimensional real FFT float functions perform two-dimensional FFTs for float data.

Two-dimensional real FFT double precision functions perform two-dimensional FFTs for double precision data.

Table 10-4: 2-D Real FFTs, Float and Double Precision Data

Float Double Precision Description
fft2d_ripx fft2d_ripdx In-place 2-D real FFT
fft2d_riptx fft2d_riptdx In-place 2-D real FFT, split format, with temp. vector
fft2d_ropx fft2d_ropdx Out-of-place 2-D real FFT
fft2d_roptx fft2d_roptdx Out-of-place 2-D real FFT, split format, with temp. vector
fft2d_zripx fft2d_zripdx In-place 2-D real FFT, split format
fft2d_zriptx fft2d_zriptdx In-place 2-D real FFT, split format, split format, with temp. vector
fft2d_zroptx fft2d_zropdx Out-of-place 2-D real FFT
fft2d_zropx fft2d_zroptdx Out-of-place 2-D real FFT, split format, split format, with temp. vector
