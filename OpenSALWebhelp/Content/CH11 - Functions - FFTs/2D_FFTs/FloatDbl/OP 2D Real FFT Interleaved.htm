Out-of-place Two-Dimensional Real Fourier Transform, Interleaved Complex Format

fft2d_ropx(), fft2d_ropdx(), fft2d_roptx(), fft2d_roptdx()

Syntax

void fft2d_ropx(
  FFT_setup *setup, /* weights array */
  SAL_f32 *A, /* input matrix (destroyed) */
  SAL_i32 i, /* row stride for A */
  SAL_i32 j, /* column stride for A */
  SAL_f32 *C, /* output matrix */

Description

The fft2d_ropx(), fft2d_ropdx(), fft2d_roptx(), and fft2d_roptdx() functions compute an out-of-place real discrete Fourier transform, from the spatial domain to the frequency domain (forward) or from the frequency domain to the spatial domain (inverse). The fft2d_ropx() and fft2d_ropdx() functions destroy the input matrix, while fft2d_roptx() and fft2d_roptdx() preserve it.

Forward transforms read real input and write packed complex output. Inverse transforms read packed complex input and write real output. As a result of packing the frequency-domain data, spatial-domain data and its equivalent frequency-domain data have the same storage requirements.

Parameters

setup

Points to a structure initialized by a prior call to FFT weights array function, fft2d_setup() . The value supplied as parameter N of the setup function must equal or exceed the value supplied as parameter N or M of the transform function, whichever is larger.

i

Specifies a stride across each row of matrix A. Specifying 1 for i processes every element across each row; specifying 2 processes every other element across each row, and so on.

j

Specifies a column stride for matrix A and, unless matrix A is a submatrix, should be allowed to default. Parameter j can be defaulted by specifying zero. The default stride equals the row stride, parameter i, multiplied by the column count, parameter N. So, if i is 1 and j is zero, every element of matrix A is processed. If i is 2 and j is zero, every other element of each row is processed.

If not zero, parameter j represents the distance between each row of matrix A. For example, if parameter j is 1024, element 1024 of matrix A equates to element (1,0), element 2048 equates to element (2,0), and so on.

k and l

Specify row and column strides for output matrix C in the same way that parameters i and j specify strides for input matrix A.

T

For fft2d_roptx() and fft2d_roptdx(), parameter T is a temporary matrix used for storing interim results. Matrix must be large enough to hold the element count as specified by parameters N and M.

log2N and log2M

Parameter log2N is the base 2 exponent of the number of columns to process for each row. Parameter log2M is the base 2 exponent of the number of rows to process. For example, to process 64 rows of 128 columns, specify 7 for parameter log2N and 6 for parameter log2M.

f

A forward/inverse directional flag that must specify one of the following:

FFT_FORWARD

Forward transform (real to packed complex).

FFT_INVERSE

Inverse transform (packed complex to real).

Results are undefined for other values of f.

For best performance, specify row strides of 1 and column strides of zero.
