Out-of-place Two-Dimensional Split Complex Fourier Transform

fft2d_zopx(), fft2d_zopdx(), fft2d_zoptx(), fft2d_zoptdx()

Syntax

void fft2d_zopx(
  FFT_setup *setup, /* weights array */
  SAL_zf32 *A, /* input matrix (destroyed) */
  SAL_i32 i, /* row stride for A */
  SAL_i32 j, /* column stride for A */
  SAL_zf32 *C, /* output matrix */
  SAL_i32 k, /* row stride for C */
  SAL_i32 l, /* column stride for C */
  SAL_ui32 log2N, /* base 2 exponent of column count */
  SAL_ui32 log2M, /* base 2 exponent of row count */
  SAL_i32 f, /* directional flag */

Description

The fft2d_zopx(), fft2d_zopdx(), fft2d_zoptx(), and fft2d_zoptdx() functions compute an out-of-place complex discrete Fourier transform of matrix A, from the spatial domain to the frequency domain (forward) or from the frequency domain to the spatial domain (inverse). The fft2d_zopx() destroys the input matrix, while fft2d_zoptx() preserves the input matrix.

Parameters

setup

Points to a structure initialized by a prior call to an FFT weights array function, fft2d_setup(). The value supplied as parameter N of the setup function must equal or exceed the value supplied as parameter N or M of the transform function, whichever is larger.

i and j

Specify strides through the input and output matrices.

The i parameter specifies a stride across each row of matrix A. Specifying 1 for i processes every element across each row; specifying 2 for i processes every other element across each row, and so on.

The j parameter specifies a column stride for matrix A, and, unless matrix A is a submatrix, should be allowed to default. Parameter j can be defaulted by specifying zero. The default stride equals the row stride, parameter i, multiplied by the column count, parameter N. So, if i is 1 and j is zero, every element of matrix A is processed. If i is 2 and j is zero, every other element of each row is processed.

If not zero, parameter j represents the distance between each row of matrix A. For example, if parameter j is 1024, element 512 equates to element (1,0) of matrix A, element 1024 equates to element (2,0), and so on.

k and l

Specify row and column strides for output matrix C in the same way that parameters i and j specify strides for input matrix A.

T

For fft2d_zoptx() and fft2d_zoptdx(), parameter T is a temporary matrix used for storing interim results. Matrix T must be large enough to hold the element count as specified by parameters N and M.

log2N and log2M

The log2N parameter is the base 2 exponent of the number of columns to process for each row. Parameter log2M is the base 2 exponent of the number of rows to process. For example, to process 64 rows of 128 columns, specify 7 for parameter log2N and 6 for parameter log2M.

f

A forward/inverse directional flag, and must specify one of the following:

FFT_FORWARD

Forward transform.

FFT_INVERSE

Inverse transform.

Results are undefined for other values of f.

For best performance, specify row strides of 1 and column strides of zero.
