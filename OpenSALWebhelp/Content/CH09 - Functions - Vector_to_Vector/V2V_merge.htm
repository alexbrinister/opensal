Vector-to-Vector Merge Operations

Vector-to-vector merge functions merge two vectors of equal length into a third vector. All of these functions operate with float data.

Table 8-3: Vector-to-Vector Merge Functions

Function Description
vabmrgx Vector arithmetic block merge
vamergx Vector arithmetic merge
vcmergx Vector compressed merge
vlmergx Vector logical merge
vpmergx Vector positive merge

Vector Arithmetic Block Merge

vabmrgx()

Syntax

void vabmrgx(
  SAL_f32 *A, /* input vector */
  SAL_f32 *B, /* input vector */
  SAL_f32 *C, /* output vector */
  SAL_i32 f, /* merge index */
  SAL_i32 n, /* length of subvectors */
  SAL_i32 p, /* number of subvectors in A */
  SAL_i32 q, /* number of subvectors in B */
  SAL_i32 eflag ); /* ESAL flag */

Description

The vabmrgx() function merges the subvectors of two input vectors.

  • Input vector A contains p subvectors of length n.
  • Input vector B contains q subvectors of length n.

Subvectors p + q of length n are output to C. Output order is determined by comparing each subvector f zero-based element, and arithmetically merging the subvectors based on this value. Although subvectors are reordered, based on a single element in each subvector, the subvector element order is unchanged.

Vector Arithmetic Merge

vamergx()

Syntax

void vamergx(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_i32 m, /* input element count for A */
  SAL_f32 *B, /* input vector */
  SAL_i32 j, /* address stride for B */
  SAL_i32 n, /* input element count for B */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 eflag ); /* ESAL flag */

Description

The vamergx() function combines vector A of length m with vector B of length n. The elements are copied to vector C in descending arithmetic order to yield m + n output values.

Vector Compressed Merge

vcmergx()

Syntax

void vcmergx(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *B, /* input vector */
  SAL_i32 j, /* address stride for B */
  SAL_f32 *C, /* input vector */
  SAL_i32 k, /* address stride for C */
  SAL_f32 *D, /* output vector */
  SAL_i32 m, /* address stride for D */
  SAL_i32 N, /* real element count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The vcmergx() function merges vectors A and B by copying an element from A or B for each element written to D. Gating vector C determines the input vector from which the output is selected. When C is zero, the output element is copied from A. When C is not zero, the output element is copied from B.

Vector Logical Merge

vlmergx()

Syntax

void vlmergx(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *B, /* input vector */
  SAL_i32 j, /* address stride for B */
  SAL_f32 *C, /* input vector */
  SAL_i32 k, /* address stride for C */
  SAL_f32 *D, /* output vector */
  SAL_i32 m, /* address stride for D */
  SAL_i32 N, /* real element count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The vlmergx() function merges vectors A and B. Vector C determines from which input vector an element is copied to vector D. Nonzero elements of C are copied from vector A, while elements of C equal to zero are copied from vector B.

Vector Positive Merge

vpmergx()

Syntax

void vpmergx(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *B, /* input vector */
  SAL_i32 j, /* address stride for B */
  SAL_f32 *C, /* input vector */
  SAL_i32 k, /* address stride for C */
  SAL_f32 *D, /* output vector */
  SAL_i32 l, /* address stride for D */
  SAL_i32 n, /* real element count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The vpmergx() function merges vectors A and B. Vector C determines from which input vector an element is copied to output vector D. Positive elements of C are copied from vector A. Negative elements of C are copied from vector B.
