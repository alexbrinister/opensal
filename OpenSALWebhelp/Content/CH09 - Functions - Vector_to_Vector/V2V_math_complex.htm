Vector-to-Vector Arithmetic Operations, Complex

Vector-to-vector arithmetic functions arithmetically combine two vectors of equal length to produce a single-vector result. Examples include:

  • Vector addition/subtraction
  • Element-by-element vector multiplication/division

Vector-to-vector complex interleaved arithmetic functions operate on complex interleaved data vectors.

Vector-to-vector complex split arithmetic functions operate on complex split data vectors.

Accumulating Autospectrum

aspecx(), zaspecx()

Syntax

void aspecx(
  SAL_cf32 *A, /* input vector */
  SAL_f32 *C, /* input-output vector */
  SAL_i32 N, /* real output count */
  SAL_i32 eflag ); /* ESAL flag */
void zaspecx(
  SAL_zf32 *A, /* input vector */
  SAL_f32 *C, /* input-output vector */
  SAL_i32 N, /* real output count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The aspecx() and zaspecx() functions multiply complex vector A by its complex conjugates, yielding the sums of the squares of the complex and real parts.

(x + iy) (x – iy) = (x*x + y*y)

The results are added to real input-output vector C. Vector C must contain valid data from previous processing or should be initialized as required before calling aspecx() or zaspecx().

Description
