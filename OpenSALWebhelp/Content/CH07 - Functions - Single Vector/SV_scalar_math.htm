Single-Vector Scalar Arithmetic Functions

Single-vector scalar arithmetic functions combine a scalar with a vector, such as multiply or divide a vector by a scalar, or add or subtract a scalar to each element of a vector.

vsdivix vsdivx vsdivdx Vector scalar divide
vsmulix vsmulx vsmuldx Vector scalar multiply
  svdivx   Scalar vector divide
  vrecipx   Vector reciprocal

Complex Vector Multiply by Complex Scalar

cvcsmlx(), cvcsmldx(), zvzsmlx(), zvzsmldx()

Syntax

void cvcsmlx(
  SAL_cf32 *A, /* input vector */
  SAL_i32 I, /* address stride for A, I=2i */
  SAL_cf32 *b, /* input scalar */
  SAL_cf32 *C, /* output vector */
  SAL_i32 K, /* address stride for C, K=2k */
  SAL_i32 n, /* complex element count */
  SAL_i32 eflag ); /* ESAL flag */
void cvcsmldx(
  SAL_cf64 *A, /* input vector */
  SAL_i32 I, /* address stride for A, I=2i */
  SAL_cf64 *b, /* input scalar */
  SAL_cf64 *C, /* output vector */
  SAL_i32 K, /* address stride for C, K=2k */
  SAL_i32 n, /* double-complex element count */
  SAL_i32 eflag ); /* ESAL flag */
void zvzsmlx(
  SAL_zf32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_zf32 *b, /* input scalar */
  SAL_zf32 *C, /* output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 n, /* complex element count */
  SAL_i32 eflag ); /* ESAL flag */
void zvzsmldx(
  SAL_zf64 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_zf64 *b, /* input scalar */
  SAL_zf64 *C, /* output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 n, /* double-complex element count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The cvcsmlx(), cvcsmldx(), zvzsmlx(), and zvzsmldx() functions multiply vector A by scalar B. To multiply by a real or double-precision scalar, store zero in the imaginary part of the source scalar B or, if the input and output vectors are contiguous (i = k = 2), call vsmulx() or vsmuldx() with a count of 2 * n.

Scalar Divide by Vector

svdivx()

Syntax

void svdivx(
  SAL_f32 *a, /* input scalar */
  SAL_f32 *B, /* input vector */
  SAL_i32 j, /* address stride for B */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 N, /* real element count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The svdivx() function divides scalar a by each element of vector B.

Vector Scalar Multiply and Scalar Add

Syntax

void vsmsax(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *b, /* input scalar */
  SAL_f32 *c, /* input scalar */
  SAL_f32 *D, /* output vector */
  SAL_i32 m, /* address stride for D */
  SAL_i32 N, /* real element count */
  SAL_i32 eflag ); /* ESAL flag */
void vsmsadx(
  SAL_f64 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f64 *b, /* input scalar */
  SAL_f64 *c, /* input scalar */
  SAL_f64 *D, /* output vector */
  SAL_i32 m, /* address stride for D */
  SAL_i32 N, /* real element count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The vsmsax() and vsmsadx() functions multiply vector A by scalar b and then add scalar c to each product.
