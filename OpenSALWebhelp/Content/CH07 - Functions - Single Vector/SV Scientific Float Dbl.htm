Single-Vector Scientific Functions, Integer, Float and Double Precision

Single-vector scientific functions compute elementary transcendental functions (sine, cosine, atan, square root, exponential, logarithm, absolute value, and so on) on each of the elements of specified input vectors.

These functions operate as follows:

  • Integer vector scientific functions operate on vectors of integer data.
  • Float vector scientific functions operate on vectors of float data.
  • Double precision vector scientific functions operate on vectors of double precision data.

Vector Convert to Power or Amplitude Decibels

vdbconx()

Syntax

void vdbconx(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* element stride for A */
  SAL_f32 *b, /* input scalar, zero reference */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* element stride for C */
  SAL_i32 N, /* element count */
  SAL_i32 f, /* power (0) or amplitude (1) flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The vdbconx() function converts inputs from vector A to its decibel equivalent, calculated in terms of power or amplitude according to flag f. As a relative reference point, the input scalar b value is considered to be at zero decibels.

Rectangular to Polar Conversion

polarx(), polardx()

Syntax

void polarx(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* address stride for A, must be even */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* address stride for C, must be even */
  SAL_i32 n, /* number of ordered pairs processed */
  SAL_i32 eflag ); /* ESAL flag */
void polardx(
  SAL_f64 *A, /* input vector */
  SAL_i32 i, /* address stride for A, must be even */
  SAL_f64 *C, /* output vector */
  SAL_i32 k, /* address stride for C, must be even */
  SAL_i32 n, /* number of ordered pairs processed */
  SAL_i32 eflag ); /* ESAL flag */

Description

The polarx() and polardx() functions convert rectangular coordinates to polar coordinates. Cartesian (x, y) pairs are read from vector A. Polar (rho, theta) pairs, where rho is the radius and theta is the angle in the range (–pi, pi), are written to vector C. Argument n indicates the number of coordinate pairs in A and C.

The polarx() and polardx() functions perform the inverse operations of rectx() and rectdx(), respectively. The rectx() and rectdx() functions convert polar coordinates to rectangular coordinates.

Polar to Rectangular Conversion

rectx(), rectdx()

Syntax

void rectx(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* address stride for A, must be even */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* address stride for C, must be even */
  SAL_i32 N, /* number of ordered pairs processed */
  SAL_i32 eflag ); /* ESAL flag */
void rectdx(
  SAL_f64 *A, /* input vector */
  SAL_i32 i, /* address stride for A, must be even */
  SAL_f64 *C, /* output vector */
  SAL_i32 k, /* address stride for C, must be even */
  SAL_i32 N, /* number of ordered pairs processed */
  SAL_i32 eflag ); /* ESAL flag */

Description

The rectx() and rectdx() functions convert polar coordinates to rectangular coordinates. Polar (rho, theta) pairs, where rho is the radius and theta is the angle in radians, are read from vector A. Cartesian (x, y) pairs are written to vector C. Argument N indicates the number of coordinate pairs in A and C.

The rectx() and rectdx() functions perform the inverse operations of polarx() or polardx(), respectively. The polarx() or polardx() functions convert rectangular coordinates to polar coordinates.
