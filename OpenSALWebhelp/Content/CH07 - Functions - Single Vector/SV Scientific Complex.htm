Single-Vector Scientific Functions, Complex

Single-vector scientific functions compute elementary transcendental functions (sine, cosine, atan, square root, exponential, logarithm, absolute value, and so on) on each of the elements of specified input vectors.

These functions operate as follows:

  • Complex interleaved vector scientific functions operate on vectors of complex interleaved data.
  • Complex split vector scientific functions operate on vectors of complex split data.

Frequency Translation by a Complex Exponential

crmixerx(), cmixerx(), zrmixerx(), zmixerx()

Syntax

void crmixerx(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* element stride for A */
  SAL_f64 *f, /* input scalar, frequency */

  SAL_cf32 *C, /* output signal vector */
  SAL_i32 k, /* element stride for C, must be even */
  SAL_i32 N, /* complex output count */
  SAL_i32 flag, /* conjugation flag */
  SAL_i32 eflag ); /* ESAL flag */

  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* element stride for A */
  SAL_f64 *f, /* input scalar, frequency */
  SAL_f64 *p, /* phase, input and output */
  SAL_zf32 *C, /* output signal vector */
  SAL_i32 k, /* element stride for C */
  SAL_i32 N, /* complex output count */

void zmixerx(
  SAL_zf32 *A, /* input vector */
  SAL_i32 i, /* element stride for A */
  SAL_f64 *f, /* input scalar, frequency */
  SAL_f64 *p, /* phase, input and output */
  SAL_zf32 *C, /* output vector */
  SAL_i32 k, /* element stride for C */
  SAL_i32 N, /* complex output count */
  SAL_i32 flag, /* conjugation flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The crmixerx(), cmixerx(), zrmixerx(), and zmixerx() functions perform frequency translation of the input signal by a complex exponential. Functionally, these functions combine vrampx(), cvexpmx() [or zvexpmx()], and cvmulx() [or zvmulx()].

The following flag options for these functions can be ORed together:

SAL_CONJUGATE_SIGNAL

Conjugate the signal vector A. Ignore if A is real.

SAL_CONJUGATE_OSCILLATOR

Conjugate the exponential.

The translation formula depends on the flag values shown in Table 6-6

Table 6-6: crmixerx(), cmixerx(), zrmixerx(), and zmixerx() Function Transform Formulas

Flag Translation Formula
None Cnk = exp(i • angle) • Ani
SAL_CONJUGATE_SIGNAL Cnk = exp(i • angle) • conj(Ani)
SAL_CONJUGATE_OSCILLATOR Cnk = exp(–i • angle) • Ani
SAL_CONJUGATE_SIGNAL
SAL_CONJUGATE_OSCILLATOR		 Cnk = exp(–i • angle) • conj(Ani)

Vector Convert to Power or Amplitude Decibels

cvdbconx(), zvdbconx()

Syntax

void cvdbconx(
  SAL_cf32 *A, /* input vector */
  SAL_i32 i, /* half-element stride for A, must be even */
  SAL_f32 *b, /* input scalar, zero reference */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* element stride for C */
  SAL_i32 N, /* element count */
  SAL_i32 f, /* power (0) or amplitude (1) flag */
  SAL_i32 eflag ); /* ESAL flag */
void zvdbconx(
  SAL_zf32 *A, /* input vector */
  SAL_i32 i, /* element stride for A */
  SAL_f32 *b, /* input scalar, zero reference */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* element stride for C */
  SAL_i32 N, /* element count */
  SAL_i32 f, /* power (0) or amplitude (1) flag */
  SAL_i32 eflag ); /* ESAL flag */

Description

The cvdbconx(), and zvdbconx() functions convert inputs from vector A to their decibel equivalents, calculated in terms of power or amplitude according to flag f. As a relative reference point, the input scalar b value is considered to be at zero decibels.

  SAL_i32 N, /* real output count */

Description

The cvphasx() and zvphasx() functions find the phase values, in radians, of complex vector A and store the results in real vector C. The results are between –pi and +pi. The sign of the result is the sign of the imaginary part of the coordinate in the input vector.
