Single-Vector Element Move Functions

Single-vector element move functions move a vector from one set of locations to another set of locations.

Vector Compress

vcmprsx()

Syntax

void vcmprsx(
  SAL_f32 *A, /* input vector */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *B, /* input vector */
  SAL_i32 j, /* address stride for B */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* address stride for C */
  SAL_i32 N, /* real input count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The vcmprsx() function compresses vector A based on the nonzero values of gating vector B. For nonzero elements of B, corresponding elements of A are sequentially copied to output vector C.

Vector Gather, Absolute Pointers

vgathrax(), vgathradx()

Syntax

void vgathrax(
  SAL_f32 **A, /* vector of pointers */
  SAL_i32 i, /* element stride for A */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* element stride for C */
  SAL_i32 N, /* real element count */
  SAL_i32 eflag ); /* ESAL flag */
void vgathradx(
  SAL_f64 **A, /* vector of pointers */
  SAL_i32 i, /* element stride for A */
  SAL_f64 *C, /* output vector */
  SAL_i32 k, /* element stride for C */
  SAL_i32 N, /* double-precision element count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The vgathrax() and vgathradx() functions use vector A as an array of absolute pointers to copy selected elements from memory to sequential locations of vector C. These functions can only be done out-of-place.

Vector Gather

vgathrx(), vgathrdx()

Syntax

void vgathrx(
  SAL_f32 *A, /* input vector */
  SAL_i32 *B, /* indices vector */
  SAL_i32 j, /* element stride for B */
  SAL_f32 *C, /* output vector */
  SAL_i32 k, /* element stride for C */
  SAL_i32 N, /* real element count */
  SAL_i32 eflag ); /* ESAL flag */
void vgathrdx(
  SAL_f64 *A, /* input vector */
  SAL_i32 *B, /* indices vector */
  SAL_i32 j, /* element stride for B */
  SAL_f64 *C, /* output vector */
  SAL_i32 k, /* element stride for C */
  SAL_i32 N, /* double-precision element count */
  SAL_i32 eflag ); /* ESAL flag */

Description

The vgathrx() and vgathrdx() functions use elements of vector B as indices to copy selected elements of vector A to sequential locations in vector C. These functions can only be done out-of-place.

When evaluating indices, the vgathrx() and vgathrdx() functions treat 1, not zero, as the first location in the input vector.

Description

The vindexx() function uses vector B as zero-based subscripts to copy selected elements of vector A to vector C. Fractional parts of vector B are ignored.
