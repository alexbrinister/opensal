Single-Vector Functions

Single-vector functions perform operations such as:

  • Filling a vector with a specified set of values.
  • Combining a scalar with a vector.
  • Computing elementary transcendental functions on each of the elements of specified input vectors.
  • Comparing each element of a given vector against a specified scalar value.
  • Moving a vector from one set of locations to another.
  • Performing (previously unspecified) operations on single vector elements.
