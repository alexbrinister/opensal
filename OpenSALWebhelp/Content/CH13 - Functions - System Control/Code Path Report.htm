Code-Path Report

sal_get_path()

Syntax

SAL_i32 sal_get_path(

SAL_i32 flags ); /* reserved. Specify zero */

Description

The sal_get_path() function returns a value indicating the version of the previously executed SAL function.

This function is not thread-safe. If more than one thread in the process invokes a SAL function, sal_get_path() can return an undefined or incorrect value.

This function is intended for development only. Do not use it in a deployed application.

The sal_get_path() function indicates which version was executed last, not necessarily which function was used to do the majority of the job.

Returns

SAL_PATH_NOTHING

No SAL function invoked.

SAL_PATH_CSAL

Executed routine from the CSAL library.

SAL_PATH_UNKNOWN

Unable to determine the executed path.
