SAL Transform Functions

SAL provides the following transform function types:

  • SAL Fast Fourier Transforms (FFTs)

    Transform data from the time domain to the frequency domain (and the reverse).

  • SAL Convolution and Correlation Functions (FCs)

    Enable single library calls to perform Fourier-based, one-dimensional convolutions and correlations.

  • SAL Image Transforms

    Perform image translations using the spinx() and warpx() functions.
