LU Decomposition

Given an n × n matrix A, the LU decomposition (LUD) consists of two n × n triangular factor matrices L (lower triangular) and U (upper triangular) so that A = L • U. By definition, the diagonal elements of the submatrix L are 1.

Figure 3-1: LU Decomposition

The matinvx(), matinvdx(), matinv_lux(), cmatinv_lux(), zmatinv_lux() SAL functions perform the LU decomposition. These functions use partial pivoting; meaning the row order in L • U can differ from the row order in A.

SAL LUD functions account for the row order change through a permutation vector output, therefore, AD[i]•j = (L • U)i•j, where D[i] is a permutation vector. These functions also implement the Reciprocal Diagonal Convention. For more information, see Reciprocal Diagonal Convention
