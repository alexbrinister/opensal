Cholesky Decomposition

The Cholesky decomposition is closely related to the LU decomposition applied to a real symmetric or complex Hermitian matrix.

Given a positive definite Hermitian matrix A, the standard Cholesky decomposition of A is the unique factorization of the form A = C • CH, where C is lower triangular with positive diagonal elements. A disadvantage of this Cholesky decomposition form is that it involves square root calculations.

SAL implements a square-root-free Cholesky decomposition. Given a positive definite Hermitian matrix A (symmetric for real data), the square-root-free Cholesky decomposition is:

A = L • D • LH

Where:

L is a lower unit triangular matrix.

- and -

Diagonal matrix D elements are squares of the diagonal matrix C elements of the standard Cholesky decomposition A = C • CH.

Figure 3-2: Square-Root-Free Cholesky Decomposition

The mat_chol_decx(), cmat_chol_decx(), zmat_chol_decx() SAL functions perform the square-root-free Cholesky decomposition. The mat_chol_solx(), cmat_chol_solx(), zmat_chol_solx() SAL functions perform a corresponding linear equation system solution.
