Terms and Definitions

To understand SAL matrix algebra functions, you must be familiar with:

  • General linear algebra definitions
  • Reciprocal diagonal convention
  • Matrix and vector norms
  • Matrix condition numbers

General Linear Algebra Definitions

The conjugate transpose AH (also known as Hermitian conjugate or adjoint) of a complex matrix A is defined by:

Re(AHij) = Re(Aji)

Im(AHij) = –Im(Aji)

where Re and Im denote the real and imaginary parts of a complex number, respectively.

A complex matrix A is Hermitian if it is equal to its Hermitian conjugate:

A = AH

If A is real, this reduces to A = AT, and the matrix is called symmetric. Here, the term “Hermitian” is preferable to “symmetric” when the matrix in question could be real symmetric or complex Hermitian.

The Hermitian inner product of two complex column M-vectors, x and y, is the product of the 1 × M row vector xH and the M × 1 column vector y:

xH · y = Σ conj(x[i]) · y[i]) i = 0 to M–1,

(1)

where conj() denotes the complex conjugate.

Real or complex vectors x and y are orthogonal if xH · y = zero. A real n × n matrix A is orthogonal if the product of the matrix and its transpose comprise the identity matrix: A · AT = I.

A set S of complex vectors is an orthonormal set if they are pairwise orthogonal, meaning:

  • xH · y = zero for all x, y in S where x ≠ y
  • Unit length (xH · x = 1 for all x in S)

A complex n × n matrix A is unitary if the product of the matrix and its Hermitian conjugate is the n × n identity matrix: A · AH = I. The set of columns (or rows) of a unitary matrix forms an orthonormal set.

An m × n matrix A is diagonal if aij = zero for i ≠ j. If p = min{m, n}, then the notation A = diag(d0, d1, d2, …, dp–1) means that A is diagonal and aii = di for i = zero to p–1.

Note 
A diagonal matrix is not necessarily square.

An m × n matrix A = (aij) is upper bidiagonal if aij = zero for all i ≠ j or i ≠ j–1.

An m × n matrix is lower bidiagonal if aij = zero for all i ≠ j or i ≠ j+1.

An n × n matrix is tridiagonal if it has non-zero elements only on the main diagonal, and the two adjacent diagonals (one above and one below).

In describing the rate of increase in operation counts of a matrix algorithm as a function of the dimension n, the O(n) indicates growth at no more than a linear rate as n increases. Given a function ƒ(n), ƒ(n) = O(n) means that there is a constant C, so that |ƒ(n)| < C· n for all n. Whereas ƒ(n) = O(n2) means that a constant C exists, so that |ƒ(n)| < C· n2 for all n.

A matrix is triangular if it is upper or lower triangular.

An n × n matrix A = (aij) is upper triangular if aij = zero for all i > j.

An n × n matrix is lower triangular if aij = 0 for all i < j.

A matrix is unit triangular if it is triangular and all diagonal elements are equal to unity (aii = 1 for i = 0 to n–1).

Reciprocal Diagonal Convention

Some of the SAL matrix decomposition algorithms (like LU decomposition and QR decomposition) use a reciprocal diagonal element convention. This means that the output diagonal elements are the reciprocals of the values that would normally be stored there. This approach eliminates multiple calculations of the diagonal element reciprocals for iterated solving of the equation A· x = b for different bs.

Matrix Condition Numbers

The matrix condition numbers discussed here are associated with the problem of solving the system of equations A • x = b, or with the closely related problem of finding the inverse matrix, A–1. There are similar notions of condition with respect to other common matrix problems (such as the eigenvalue problem), but these are beyond the scope of this topic. Also, this topic deals only with relative (not absolute) condition, so the qualifier “relative” is sometimes omitted.

The p-norm relative condition number of a square non-singular matrix A is:

kp(A) = ||A||p· ||A–1||p

(6)

If written without the p subscript (that is, k(A) = ||A|| • ||A–1||), the 2-norm is assumed. If A is singular, then, by convention, kp(A) = ∞.

The componentwise relative condition number (also known as Bauer-Skeel condition) of matrix A is defined as:

cond(A) = |||A–1| • |A|||∞ ,

(7)

where |A| is the matrix of absolute values, |aij|. You can define a p-norm componentwise condition number, however, the case of p = ∞ only is considered.

The 2-norm condition number can also be generalized to non-square matrices using the singular value decomposition (SVD) (see For more information, see Singular Value Decomposition on page 1.Singular Value Decomposition) or the matrix pseudo inverse. The pseudo inverse A+ of an m × n matrix A with SVD A = U• ∑ • VH and rank r is the n × m matrix A+ = V• ∑+• UH, where ∑+ is the n × m diagonal matrix of inverse singular values:

∑+ = diag(1/σ0,…, 1/σr–1,…, 0,…, 0).

(8)

If A is a non-singular square matrix, then A+ = A–1. In more general terms, if m × n and A is a full-rank matrix (rank(A) = n), then A+ = (AH • A)–1 • AH.

In terms of the SVD, the 2-norm condition number of a full-rank rectangular (m × n) matrix A with m ≥ n and rank(A) = n (full rank) is defined as k2(A) = σ1/σn–1 .

For an arbitrary p-norm, the condition number of a rectangular matrix can be defined in terms of the pseudo-inverse: kp(A) = ||A||p • ||A+||p. For p = 2, this formula gives the same condition number values as Equation (8), in terms of singular values.

If A is an m × n matrix with m ≥ n and rank (A) = n, then its 2-norm, k2(A), condition number, or Frobenius norm, kF(A), condition number is contained entirely in its R-factor. That is, if A = Q • R is a (thin) QR factorization of A, then kp(A) = kp(R) for p = 2 or p = F.

If U is an n × n unitary matrix, then k2(U) = 1.
