Symmetric Matrix Eigenvalues and Eigenvectors

Given a matrix A, the eigenvalue problem is to find the scalars l (the eigenvalues) and corresponding nonzero vectors z (the eigenvectors) so that A • z = λ • z. If the matrix A is real symmetric, this problem is called a Symmetric Eigenvalue Problem (SEP). If A is a Hermitian matrix, then the problem of finding the eigenvalues and eigenvectors is called the Hermitian Eigenvalue Problem (HEP). The eigenvalues of a Hermitian matrix are always real.

A spectral factorization (or unitary diagonalization, or eigendecomposition) A = V • Λ • VH exists for any Hermitian n × n matrix A, where V is unitary and Λ is a diagonal matrix with diagonal elements equal to the eigenvalues of A. I, the factorization can be done so that the eigenvalues appear in descending order; for example, Λ = diag(λ0, λ1, …,λn–1) with λ0 ≥ λ1 ≥ … ≥ λn–1. Note that the columns of V are the orthonormal eigenvectors of A.

The zmatherm_eigx() SAL function calculates eigenvalues and corresponding eigenvectors of an n × n Hermitian matrix. The resulting eigenvalues define the diagonal matrix Λ, and the corresponding eigenvectors define the n × n orthonormal matrix V.
