Makefiles

To edit your makefiles to work with MCSAL, use the following procedures.

Makefile for C++ or C Programs

For linking, the MCSAL and SAL libraries must be in the following order:

  1. libmcsal.a
  2. libsal.a
  3. libcsal.a

Because MCSAL uses pthreads to create and join threads, the link line must include the POSIX® thread library (-lpthread). For timing, include the real time library (-lrt).

-lmcsal -lsal -lcsal -lrt -lpthread

Makefile for C Programs

Because MCSAL is written in C++, you must add the C++ standard library (-lstdc++) to the link line.

-lmcsal -lsal -lcsal -lrt -lpthread -lstdc++
