Eigenvalues and Eigenvectors of Hermitian Matrix

zmatherm_eigx()

A = V * D * VH

Syntax

SAL_i32 zmatherm_eigx(
  SAL_zf32 *A, /* n x n Hermitian matrix. Destroyed on exit */
  SAL_i32 Atcols, /* matrix A column stride (>=n) */
  SAL_f32 *D, /* eigenvalues. Output vector of length n */
  SAL_zf32 *V, /* eigenvectors. Output, stored as rows */
  SAL_i32 Vtcols, /* matrix V column stride (>=n) */
  SAL_i32 n, /* matrix A and V row and column count */
  SAL_eig_tune *eig_tune, /* reserved, specify NULL. */
  SAL_f32 *cond_num, /* condition number, or NULL */
  SAL_i32 *index_vector, /* permutation vector of eigenvalue indices */
  SAL_i32 flag, /* reserved, specify zero */
  SAL_i32 eflag ); /* ESAL flag */

Description

The zmatherm_eigx() function calculates eigenvalues and eigenvectors of a Hermitian matrix.

Parameters

Atcols

Specifies a column stride of input matrix A. Atcols must be ≥ n. If Atcols = n, successive rows of A are contiguous in memory. If Atcols > n, A is a submatrix.

Vtcols

Specifies a column stride through output matrix V, which contains an orthonormal set of eigenvectors of A as rows. Specify V = NULL to skip eigenvector calculation (calculate eigenvalues only).

cond_num

The condition number output pointer. Specify cond_num = NULL to skip condition number calculation.

index_vector

The pointer to the returned permutation vector containing indices of eigenvalues in order of decreasing absolute value. Specify index_vector = NULL to skip the eigenvalue sort.

Returns

SAL_SUCCESS

Success. Otherwise, the function failed to converge within the maximum number of iterations (defined internally).
