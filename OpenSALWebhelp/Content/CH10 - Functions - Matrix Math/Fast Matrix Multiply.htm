Fast Matrix Multiply

mmulfx(), mmulfdx()

Syntax

void mmulfx(
  SAL_f32 *A, /* input matrix */
  SAL_f32 *B, /* input matrix */
  SAL_f32 *C, /* output matrix */
  SAL_i32 M, /* row count for A and C, must be even */
  SAL_i32 N, /* column count for B and C, must be even */
  SAL_i32 P, /* col. count for A, row count for B, must be even */
  SAL_i32 eflag ); /* ESAL flag */
void mmulfdx(
  SAL_f64 *A, /* input matrix */
  SAL_f64 *B, /* input matrix */
  SAL_f64 *C, /* output matrix */
  SAL_i32 M, /* row count for A and C, must be even */
  SAL_i32 N, /* column count for B and C, must be even */
  SAL_i32 P, /* col. count for A, row count for B, must be even */
  SAL_i32 eflag ); /* ESAL flag */

Description

The mmulfx() and mmulfdx() functions multiply M x P matrix A by P x N matrix B and store the results in M x N matrix C. To improve performance, the mmulfx(), and mmulfdx() functions do not accept address strides. Instead, the address strides default to 1. These functions can only be done out-of-place.
