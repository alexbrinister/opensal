Forward-Backward Substitution (for LU Decomposition)

matfbsx(), matfbsdx()

Syntax

void matfbsx(
  SAL_f32 *A, /* LU-decomposed input matrix */
  SAL_f32 *B, /* input vector of reciprocals */
  SAL_i32 *P, /* input permutation vector */
  SAL_f32 *C, /* input vector */
  SAL_f32 *D, /* output vector */
  SAL_i32 n, /* column count and row count for A, element count for B, C, D, and P */
  SAL_i32 eflag ); /* ESAL flag */
void matfbsdx(
  SAL_f64 *A, /* LU-decomposed input matrix */
  SAL_f64 *B, /* input vector of reciprocals */
  SAL_i32 *P, /* input permutation vector */
  SAL_f64 *C, /* input vector */
  SAL_f64 *D, /* output vector */
  SAL_i32 n, /* column count and row count for A, element count for B, C, D, and P */
  SAL_i32 eflag ); /* ESAL flag */

Description

The matfbsx() function uses forward-backward substitution to solve a set of equations in the form Mx = B, where:

  • M represents input matrix A
  • B represents input vector C
  • x represents output vector D

Matrix A is a decomposed input matrix created by matludx(), cmatludx(), or zmatludx(). Vector P is a permutation vector created by matludx(), cmatludx(), or zmatludx(). Vector B is a vector of reciprocals, one for each element of the matrix A main diagonal, created by the caller. Therefore:

B[m] = 1.0/A[m,m] for m ={0, n-1}

Vector B can be created from matrix A using cvrcipx(), vrecipx(), or zvrcipx(), as follows:

vrecipx (A, n+1, B, 1, n, SAL_NN);

This function can be done out-of-place only.
