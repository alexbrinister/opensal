Description

The mat_mgs_solrx() and mat_mgs_solrdx() functions solve triangular matrix systems of n equations in n unknowns Rx = b produced by the MGS QR decomposition functions. This function accepst an n x n input triangular matrix R and a right-hand side vector of known values B, and solves for the unknowns using back-substitution.

The mat_mgs_solrhrx() and mat_mgs_solrhrdx() functions solve triangular matrix systems of n equations in n unknowns RHRx = b produced by the MGS QR decomposition functions. These functions also accept an n x n input matrix R and a right-hand side vector of known values B, but solve for the unknowns by solving:

RHy = b for y by forward substitution

- then -

Rx = y for x by back substitution

Matrix RH is the conjugate transpose of a complex matrix R, also known as the Hermitian conjugate. Both halves of the R matrix (stored in the special packed format described in LU Decomposition ) are used in this case.

All of these MGS QR solvers require the special R matrix format. The mat_mgs_decx() and mat_mgs_decdx() MGS QRD functions produce the R matrix (stored in the special packed format described in LU Decomposition ) that is required by the solver functions.

Parameters

R

Input matrix.

R_tcols

Matrix R total number of columns.

B

Input vector of right-hand side values.

X

Vector of unknowns (solutions to R · x = b or RH · R · x = b).

n

Number of rows and columns in R.

flag

Specify one of the following solver options:

SAL_NORMAL_SOLVER

Solve the R · x = b equation. (default)

SAL_TRANSPOSE_SOLVER

Solve the RH · R · x = b equation.

Returns

SAL_SUCCESS

Successful execution.

Other return values

A floating point exception is detected.
