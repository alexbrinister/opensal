Submatrix Transpose

cmtrans2x(), cmtrans2dx(), zmtrans2x(), zmtrans2dx()

Syntax

void cmtrans2x(
  SAL_cf32 *A, /* input submatrix */
  SAL_cf32 *C, /* output submatrix */
  SAL_i32 M, /* row count for C, column count for A */
  SAL_i32 N, /* column count for C, row count for A */
  SAL_i32 p, /* column count for matrix containing A */
  SAL_i32 q, /* column count for matrix containing C */
  SAL_i32 eflag ); /* ESAL flag */
void cmtrans2dx(
  SAL_cf64 *A, /* input submatrix */
  SAL_cf64 *C, /* output submatrix */
  SAL_i32 M, /* row count for C, column count for A */
  SAL_i32 N, /* column count for C, row count for A */
  SAL_i32 p, /* column count for matrix containing A */
  SAL_i32 q, /* column count for matrix containing C */
  SAL_i32 eflag ); /* ESAL flag */
void zmtrans2dx(
  SAL_zf64 *A, /* input submatrix */
  SAL_zf64 *C, /* output submatrix */
  SAL_i32 M, /* row count for C, column count for A */
  SAL_i32 N, /* column count for C, row count for A */
  SAL_i32 p, /* column count for matrix containing A */
  SAL_i32 q, /* column count for matrix containing C */
  SAL_i32 eflag ); /* ESAL flag */
void zmtrans2x(
  SAL_zf32 *A, /* input submatrix */
  SAL_zf32 *C, /* output submatrix */
  SAL_i32 M, /* row count for C, column count for A */
  SAL_i32 N, /* column count for C, row count for A */
  SAL_i32 p, /* column count for matrix containing A */
  SAL_i32 q, /* column count for matrix containing C */
  SAL_i32 eflag ); /* ESAL flag */

Description

The cmtrans2x(), cmtrans2dx(), zmtrans2x(), and zmtrans2dx() functions transpose the N x M source matrix pointed to by A to the M x N destination matrix pointed to by C. The P elements separate the start of adjacent rows in matrix A, and Q elements separate the start of adjacent rows in matrix C.

Argument P must be ≥ M. If P equals M, rows in source matrix A are contiguous. If P exceeds M, rows in matrix A are not contiguous, making matrix A a submatrix of a larger matrix of N rows by P columns.

Argument Q must be ≥ N. If Q equals N, rows in destination matrix C are contiguous. If Q exceeds N, rows in C are not contiguous, making C a submatrix of a larger matrix of M rows by Q columns.
