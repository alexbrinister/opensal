Single Precision Complex Matrix Multiply and Subtract

cmmsx(), zmmsx()

Syntax

void cmmsx(
  SAL_cf32 *A, /* input matrix */
  SAL_i32 I, /* address stride for A, I=2i */
  SAL_cf32 *B, /* input matrix */
  SAL_i32 J, /* address stride for B, J=2j */
  SAL_cf32 *C, /* input matrix */
  SAL_i32 K, /* address stride for C, K=2k */
  SAL_cf32 *D, /* output matrix */
  SAL_i32 L, /* address stride for D, L=2l */
  SAL_i32 M, /* row count for A, C, and D */
  SAL_i32 N, /* column count for B, C, and D */
  SAL_i32 P, /* column count for A, row count for B */
  SAL_i32 eflag ); /* ESAL flag */
void zmmsx(
  SAL_zf32 *A, /* input matrix */
  SAL_i32 i, /* address stride for A */
  SAL_zf32 *B, /* input matrix */
  SAL_i32 j, /* address stride for B */
  SAL_zf32 *C, /* input matrix */
  SAL_i32 k, /* address stride for C */
  SAL_zf32 *D, /* output matrix */
  SAL_i32 l, /* address stride for D */
  SAL_i32 M, /* row count for A, C, and D */
  SAL_i32 N, /* column count for B, C, and D */
  SAL_i32 P, /* column count for A, row count for B */
  SAL_i32 eflag ); /* ESAL flag */

Description

The cmmsx() and zmmsx() functions multiply matrix A by matrix B, subtract matrix C from the product, and store the result to matrix D.
