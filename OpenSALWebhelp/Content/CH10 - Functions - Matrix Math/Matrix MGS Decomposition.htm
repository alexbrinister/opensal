Description

The cmat_mgs_decx(), cmat_mgs_decx(), zmat_mgs_decx(), and zmat_mgs_decdx() functions perform thin QR matrix decomposition using the Modified Gram-Schmidt (MGS) method, which is a factorization of an m x n matrix A, with m >= n, into an m x n matrix Q with orthonormal columns, times an upper-triangular matrix R.

If you are solving a linear system R · x = b, then you can use these functions and then proceed with the MGS QR solution functions, cmat_mgs_solrx(), cmat_mgs_solrdx(), cmat_mgs_solrhrx(), cmat_mgs_solrhrdx(), zmat_mgs_solrx(), zmat_mgs_solrdx(), zmat_mgs_solrhrx(), zmat_mgs_solrhrdx().

Parameters

AT

Input matrix, the transpose of an m x n matrix A (A = Q · R), with m >= n and full column rank. On exit, the input storage location contains a transpose of an m x n matrix Q with orthonormal columns.

AT_tcols

Matrix AT column element stride.

R

Special format output matrix. Each diagonal element is replaced by its reciprocal. The zeros below the diagonal are replaced by the corresponding elements above the diagonal, making the R matrix symmetric. The diagonal elements of R are presented as reciprocals to expedite solving a linear system R · x = b for multiple right-hand sides b.

For example, if n = 4, the theoretical R matrix is:

  r00 r01 r02 r03
  0 r11 r12 r13
  0 0 r22 r23
  0 0 0 r33

The corresponding 4 x 4 matrix in special R matrix format is:

  1/r00 r01 r02 r03
  r01 1/r11 r12 r13
  r02 r12 1/r22 r23
  r03 r13 r23 1/r33

R_tcols

Matrix R column element stride.

m

Matrix A and Q row count, equal to the matrix AT column count.

n

Matrix A and Q column count, equal to the matrix AT row count.

rcond

Reserved; specify NULL.

v

Reserved; specify NULL.

flag

Returns

SAL_SUCCESS

Successful execution.

Other return values

A floating point exception is detected. The resulting matrix is unusable.
