Matrix Transpose

mtransx(), mtransdx()

Syntax

void mtransx(
  SAL_f32 *A, /* input matrix */
  SAL_i32 i, /* address stride for A */
  SAL_f32 *C, /* output matrix */
  SAL_i32 k, /* address stride for C */
  SAL_i32 M, /* row count for C, column count for A */
  SAL_i32 N, /* column count for C, row count for A */
  SAL_i32 eflag ); /* ESAL flag */
void mtransdx(
  SAL_f64 *a, /* input matrix */
  SAL_i32 i, /* address stride for a */
  SAL_f64 *C, /* output matrix */
  SAL_i32 k, /* address stride for C */
  SAL_i32 M, /* row count for C, column count for A */
  SAL_i32 N, /* column count for C, row count for A */
  SAL_i32 eflag ); /* ESAL flag */

Description

The mtransx() and mtransdx() functions create transposed matrix C from source matrix A. Arguments M and N specify the number of elements in each row or column. These functions can be done out-of-place only.
