Matrix Cholesky Solver

mat_chol_solx(), mat_chol_soldx()

Syntax

SAL_i32 mat_chol_solx(
  SAL_f32 *LDU, /* input packed LDU factor matrix */
  SAL_i32 LDU_tcols, /* matrix LDU column stride */
  SAL_f32 *B, /* right-hand side vector, input */
  SAL_f32 *X, /* solution vector, output */
  SAL_i32 n, /* row/col count for LDU, element count for B and X */
  SAL_i32 flag, /* reserved, specify 0 */
  SAL_i32 eflag ); /* ESAL flag */
SAL_i32 mat_chol_soldx(
  SAL_f64 *LDU, /* input packed LDU factor matrix */
  SAL_i32 LDU_tcols, /* matrix LDU column stride */
  SAL_f64 *B, /* right-and side vector, input */
  SAL_f64 *X, /* solution vector, output */
  SAL_i32 n, /* row/col count for LDU, element count for B and X */
  SAL_i32 flag, /* reserved, specify 0 */
  SAL_i32 eflag ); /* ESAL flag */

Description

The mat_chol_solx() and mat_chol_soldx() functions solve systems of linear equations LDU * X = B using the output LDU from the corresponding Cholesky decomposition function, mat_chol_decx() or mat_chol_decdx(), with input vectors.

Parameters

LDU

Input matrix containing triangular factors L and U, and diagonal matrix D of a Cholesky decomposition (A = L · D · U) in the special packed format described in LU Decomposition. Use the mat_chol_decx() or mat_chol_decdx() function to obtain the input.

LDU_tcols

Column element stride for LDU.

B

Input vector of right-side quantities (A · x = b, where A = L · D · U)

X

Vector of unknowns (solutions). Output.

n

Row and column count for LDU, element count for X and B.

Returns

SAL_SUCCESS

Successful execution.

Other return values

A floating point exception is detected.
