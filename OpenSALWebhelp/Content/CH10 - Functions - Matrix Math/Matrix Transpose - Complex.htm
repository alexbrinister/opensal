Matrix Transpose

cmtransx(), cmtransdx(), zmtransx(), zmtransdx()

Syntax

void cmtransx(
  SAL_cf32 *A, /* input matrix */
  SAL_i32 I, /* address stride for A, I=2i */
  SAL_cf32 *C, /* output submatrix */
  SAL_i32 K, /* address stride for C, K=2k */
  SAL_i32 M, /* row count for C, column count for A */
  SAL_i32 N, /* column count for C, row count for A */
  SAL_i32 eflag ); /* ESAL flag */
void cmtransdx(
  SAL_cf64 *A, /* input matrix */
  SAL_i32 I, /* address stride for A, I=2i */
  SAL_cf64 *C, /* output submatrix */
  SAL_i32 K, /* address stride for C, K=2k */
  SAL_i32 M, /* row count for C, column count for A */
  SAL_i32 N, /* column count for C, row count for A */
  SAL_i32 eflag ); /* ESAL flag */
void zmtransx(
  SAL_zf32 *A, /* input matrix */
  SAL_i32 i, /* address stride for A */
  OMPLEX_SPLIT *C, /* output submatrix */
  SAL_i32 k, /* address stride for C */
  SAL_i32 M, /* row count for C, column count for A */
  SAL_i32 N, /* column count for C, row count for A */
  SAL_i32 eflag ); /* ESAL flag */
void zmtransdx(
  SAL_zf64 *A, /* input matrix */
  SAL_i32 i, /* address stride for A */
  SAL_zf64 *C, /* output submatrix */
  SAL_i32 k, /* address stride for C */
  SAL_i32 M, /* row count for C, column count for A */
  SAL_i32 N, /* column count for C, row count for A */
  SAL_i32 eflag ); /* ESAL flag */

Description

The cmtransx(), cmtransdx(), zmtransx(), and zmtransdx() functions create transposed matrix C from source matrix A. Arguments M and N specify the number of elements in each row or column. These functions can be done out-of-place only.
