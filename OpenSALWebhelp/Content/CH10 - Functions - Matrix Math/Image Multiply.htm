Image Multiply

cimulx(), cimuldx(), zimulx(), zimuldx()

Syntax

void cimulx(
  SAL_cf32 *A, /* packed input matrix */
  SAL_cf32 *B, /* packed input matrix */
  SAL_cf32 *C, /* packed output matrix */
  SAL_i32 M, /* matrix row count, 16 <= m <= 1024 */
  SAL_i32 N, /* matrix column count (complex), 8 <= n <= 1024 */
  SAL_i32 f, /* conjugate flag (1 normal, -1 conjugate) */
  SAL_i32 eflag ); /* ESAL flag */
void cimuldx(
  SAL_cf64 *A, /* input image */
  SAL_cf64 *B, /* input image */
  SAL_cf64 *C, /* output image */
  SAL_i32 M, /* image row count, 16 <= m <= 1024 */
  SAL_i32 N, /* image column count, 8 <= n <= 512 */
  SAL_i32 f, /* conjugate flag (1 normal, -1 conjugate) */
  SAL_i32 eflag ); /* ESAL flag */
void zimulx(
  SAL_zf32 *A, /* packed input matrix */
  SAL_zf32 *B, /* packed input matrix */
  SAL_zf32 *C, /* packed output matrix */
  SAL_i32 M, /* matrix row count, 16 <= m <= 1024 */
  SAL_i32 N, /* matrix col. count (complex), 8 <= n <= 1024 */
  SAL_i32 f, /* conjugate flag (1 normal, -1 conjugate) */
  SAL_i32 eflag ); /* ESAL flag */
void zimuldx(
  SAL_zf64 *A, /* input image */
  SAL_zf64 *B, /* input image */
  SAL_zf64 *C, /* output image */
  SAL_i32 M, /* image row count, 16 <= m <= 1024 */
  SAL_i32 N, /* image column count, 8 <= n <= 512 */
  SAL_i32 f, /* conjugate flag (1 normal, -1 conjugate) */
  SAL_i32 eflag ); /* ESAL flag */

Description

The cimulx(), cimuldx(), zimulx(), and zimuldx() functions multiply two images left in packed frequency-domain format by fft2d_ripx(), fft2d_riptx(), fft2d_ropx(), or fft2d_roptx().

Parameters

f

Multiplication flag. Assign the following values:

  • 1 Normal multiplication
  • –1 Multiplication by the conjugated elements of B

Results are undefined for other values of f.

M

The number of rows.

N

The number of complex points per row (complex columns).
