Matrix Invert, Float and Double

matinvx(), matinvdx(), matinv_lux()

Syntax

void matinvx(
  SAL_f32 *A, /* input matrix */
  SAL_f32 *C, /* output matrix */
  SAL_i32 N, /* column count and row count for A, <= 512 */
  SAL_i32 eflag ); /* ESAL flag */
void matinvdx(
  SAL_f64 *A, /* input matrix */
  SAL_f64 *C, /* output matrix */
  SAL_i32 N, /* column count and row count for A, <= 512 */
  SAL_i32 eflag ); /* ESAL flag */
void matinv_lux(
  SAL_f32 *A, /* input matrix */
  SAL_f32 *C, /* output matrix */
  SAL_i32 *B, /* input permutation vector of size N */
  SAL_i32 N, /* column and row count for A and C; length of B. N <= 512 */
  SAL_i32 eflag ); /* ESAL flag */

Description

The matinvx() and matinvdx() functions invert a square matrix using Gaussian elimination. Both functions destroy the source matrix A. If source matrix A is singular or nearly singular, these functions might cause a floating-point exception. These functions can be done out-of-place only.

The matinv_lux(), cmatinv_lux(), and zmatinv_lux() functions invert a square matrix using LU decomposition. Input matrix A is destroyed. Vector B is a temporary permutation vector used by the function. These functions can be done out-of-place only.
