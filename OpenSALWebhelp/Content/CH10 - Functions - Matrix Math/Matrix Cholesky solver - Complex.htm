Matrix Cholesky Solver

cmat_chol_solx(), cmat_chol_soldx(), zmat_chol_solx(), zmat_chol_soldx()

Syntax

 

SAL_i32 cmat_chol_solx(
  SAL_cf32 *LDU, /* input packed LDU factor matrix */
  SAL_i32 LDU_tcols, /* matrix LDU column stride */
  SAL_cf32 *B, /* right hand side vector, input */
  SAL_cf32 *X, /* solution vector, output */
  SAL_i32 n, /* row/col count for LDU, element count for B and X */
  SAL_i32 flag, /* reserved, specify 0 */
  SAL_i32 eflag ); /* ESAL flag */
SAL_i32 cmat_chol_soldx(
  SAL_cf64 *LDU, /* input packed LDU factor matrix */
  SAL_i32 LDU_tcols, /* matrix LDU column stride */
  SAL_cf64 *B, /* right hand side vector, input */
  SAL_cf64 *X, /* solution vector, output */
  SAL_i32 n, /* row/col count for LDU, element count for B and X */
  SAL_i32 flag, /* reserved, specify 0 */
  SAL_i32 eflag ); /* ESAL flag */
SAL_i32 zmat_chol_solx(
  SAL_zf32 *LDU, /* input packed LDU factor matrix */
  SAL_i32 LDU_tcols, /* matrix LDU column stride */
  SAL_zf32 *B, /* right hand side vector, input */
  SAL_zf32 *X, /* solution vector, output */
  SAL_i32 n, /* row/col count for LDU, element count for B and X */
  SAL_i32 flag, /* reserved, specify 0 */
  SAL_i32 eflag ); /* ESAL flag */
SAL_i32 zmat_chol_soldx(
  SAL_zf64 *LDU, /* input packed LDU factor matrix */
  SAL_i32 LDU_tcols, /* matrix LDU column stride */
  SAL_zf64 *B, /* right hand side vector, input */
  SAL_zf64 *X, /* solution vector, output */
  SAL_i32 n, /* row/col count for LDU, element count for B and X */
  SAL_i32 flag, /* reserved, specify 0 */
  SAL_i32 eflag ); /* ESAL flag */

Description

The cmat_chol_solx(), cmat_chol_soldx(), zmat_chol_solx(, and zmat_chol_soldx() functions solve systems of linear equations LDU * X = B using the output LDU from the corresponding Cholesky decomposition functions—cmat_chol_decx(), cmat_chol_decdx(), zmat_chol_decx(), and zmat_chol_decdx()—with input vectors.

Parameters

LDU

Input matrix containing triangular factors L and U, and diagonal matrix D of a Cholesky decomposition (A = L · D · U) in the special packed format described in LU Decomposition. Use the cmat_chol_solx(), mat_chol_soldx(), zmat_chol_solx(), and zmat_chol_soldx() functions to obtain the input.

LDU_tcols

Column element stride for LDU.

B

Input vector of right-side quantities (A · x = b, where A = L · D · U)

X

Vector of unknowns (solutions). Output.

n

Row and column count for LDU, element count for X and B.

Returns

SAL_SUCCESS

Successful execution.

Other return values

A floating point exception is detected.
