Matrix Invert, Complex

cmatinv_lux(), zmatinv_lux()

Syntax

void cmatinv_lux(
  SAL_cf32 *A, /* input matrix */
  SAL_cf32 *C, /* output matrix */
  SAL_i32 *B, /* input permutation vector of size N */
  SAL_i32 N, /* column and row count for A and C; length of B. N <= 256 */
  SAL_i32 eflag ); /* ESAL flag */
void zmatinv_lux(
  SAL_zf32 *A, /* input matrix */
  SAL_zf32 *C, /* output matrix */
  SAL_i32 *B, /* input permutation vector of size N */
  SAL_i32 N, /* column and row count for A and C; length of B. N <= 256 */
  SAL_i32 eflag ); /* ESAL flag */

Description

The cmatinv_lux() and zmatinv_lux() functions invert a square matrix using LU decomposition. Input matrix A is destroyed. Vector B is a temporary permutation vector used by the function. These functions can be done out-of-place only.
