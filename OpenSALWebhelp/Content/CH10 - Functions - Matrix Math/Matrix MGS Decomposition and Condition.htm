Matrix MGS Decomposition and Condition

mat_mgs_decx(), mat_mgs_decdx()

A = Q· R

Syntax

SAL_i32 mat_mgs_decx(
  SAL_f32 *AT, /* input/output matrix */
  SAL_i32 AT_tcols, /* matrix AT column element stride */
  SAL_f32 *R, /* n x n output matrix */
  SAL_i32 R_tcols, /* matrix R column element stride */
  SAL_i32 m, /* matrix A column count */
  SAL_i32 n, /* matrix A row count */
  SAL_f32 *rcond, /* ptr to estimate of cond.number reciprocal */
  SAL_f32 *v, /* minimizing vector for A */
  SAL_i32 flag, /* type of condition estimate */
  SAL_i32 eflag ); /* ESAL flag */
SAL_i32 mat_mgs_decdx(
  SAL_f64 *AT, /* input/output matrix */
  SAL_i32 AT_tcols, /* matrix AT column element stride */
  SAL_f64 *R, /* n x n output matrix */
  SAL_i32 R_tcols, /* matrix R column element stride */
  SAL_i32 m, /* matrix A column count */
  SAL_i32 n, /* matrix A row count */
  SAL_f32 *rcond, /* ptr to estimate of cond.number reciprocal */
  SAL_f64 *v, /* minimizing vector for A */
  SAL_i32 flag, /* type of condition estimate */
  SAL_i32 eflag ); /* ESAL flag */

Description

The mat_mgs_decx() znd mat_mgs_decdx() functions perform thin QR matrix decomposition using the Modified Gram-Schmidt (MGS) method, which is a factorization of an m x n matrix A, with m ≥ n, into an m x n matrix Q with orthonormal columns, times an upper-triangular matrix R.

If you are solving a linear system R · x = b, you can use this function and then proceed with the MGS QR solution functions, mat_mgs_solrx(), mat_mgs_solrx(), mat_mgs_solrhrx(), and mat_mgs_solrhrx().

Parameters

AT

Input matrix, the transpose of an m x n matrix A (A = Q · R), with m >= n and full column rank. On exit, the input storage location contains a transpose of an m x n matrix Q with orthonormal columns.

AT_tcols

Matrix AT column element stride.

R

Special format output matrix. Each diagonal element is replaced by its reciprocal. The zeros below the diagonal are replaced by the corresponding elements above the diagonal, making the R matrix symmetric. The diagonal elements of R are presented as reciprocals to expedite solving a linear system R · x = b for multiple right-hand sides b.

For example, if n = 4, the theoretical R matrix is:

  r00 r01 r02 r03
  0 r11 r12 r13
  0 0 r22 r23
  0 0 0 r33

The corresponding 4 x 4 matrix in special R matrix format is:

  1/r00 r01 r02 r03
  r01 1/r11 r12 r13
  r02 r12 1/r22 r23
  r03 r13 r23 1/r33

R_tcols

Matrix R column element stride.

m

Matrix A and Q row count, equal to the matrix AT column count.

n

Matrix A and Q column count, equal to the matrix AT row count.

rcond

Pointer to the estimate of reciprocal of 1-norm or infinity-norm condition number of A (for m = n) or R (for m > n), or reciprocal of componentwise condition number of A, depending on the flag setting.

v

Minimizing vector, sometimes referred to as an “approximate null vector” when A is ill-conditioned, if flag != SAL_COND_EST_NONE.

flag

Specify one of the following condition estimate options:

SAL_COND_EST_NONE

Do not compute condition estimate (default). Use this option to get best performance if you know that the matrix is well-conditioned.

When this option is selected, no memory has to be allocated for the corresponding outputs, rcond and v, and the pointers are permitted to be NULL.

SAL_COND_EST_1_NORM

Estimate 1-norm condition number.

SAL_COND_EST_INF_NORM

Estimate infinity-norm condition number.

SAL_COND_EST_COMPONENT

Estimate infinity-norm componentwise (Bauer-Skeel) condition number.

Returns

SAL_SUCCESS

Successful execution.

Other return values

A floating point exception is detected. The resulting matrix is unusable.
