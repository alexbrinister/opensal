Conventions

This publication might use the following typographic, syntactic, and command line conventions.

Table 12-3: Typographic Conventions

Convention Description
/s/node/setup.exe Function, file, and path names
argument_value User-specified value
MC_ENV Environment variables important term Indicates a new term or glossary entry
Ctrl+Z,Open File dialog box Keystrokes and user interface elements
Start > Settings User interface menu command selection

Table 12-4: Syntactic Conventions

Convention Description
{ } Indicates a required element. In this group, select one element only
[ ] Indicates an optional element
{choice | choice} Indicates that you must choose one item
[choice | choice] Indicates that you can choose either item
… [, …] [; …] Indicates that the preceding syntax element can be repeated any number of times, with a separator between elements
0x200 Represents a hexadecimal value
0b1001 Represents a binary value
Ellipsis (…) Indicates omitted text in a list or example
