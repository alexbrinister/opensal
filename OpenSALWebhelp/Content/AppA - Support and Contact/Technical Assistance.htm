Technical Assistance

So that we can better help you, before you contact Mercury Customer Support, try to isolate the problem and gather information.

  1. Complete any troubleshooting procedures in your product documentation.
  2. On systems, run the system diagnostics (if applicable) and record any information provided.
  3. Use Mercury’s online services for help with installation and troubleshooting procedures (https://customers.mc.com/self_solve/index.cfm)

If the preceding steps do not resolve the problem, contact Mercury for technical assistance.

Before contacting Mercury Customer Support, gather the following information:

  • Return Material Authorization number (if applicable)
  • Hardware and firmware configuration
  • Hardware serial numbers and firmware versions
  • Network information
  • Operating system and all software program versions/revisions

When calling by telephone:

  • Make sure that the product documentation is at hand.

  • If possible, turn on your system before you contact Mercury and call from a telephone at or near the computer.

    This allows you to walk-through the problem with the support team member, who might ask you to enter keyboard commands, provide detailed information, or perform troubleshooting steps.
